オーバーヘッドとは、本来行いたい処理そのものではないが、その処理を実行するために必要となる余分な処理や時間。

| 場面 | オーバーヘッドの例 |
| --- | --- |
| OS | プロセスの切替、メモリ管理 |
| 通信 | ヘッダの付加、確認応答、再送制御 |
| データベース | ロック管理、ログの記録 |
| 仮想化 | 仮想マシンを管理する処理 |
| データ圧縮 | 圧縮・展開に必要な処理 |

オーバーヘッドが大きいほど、本来の処理に使える時間が減り、性能が低下する場合がある。

## ターンアラウンドタイムとの関係

```text
ターンアラウンドタイム
= 待ち時間 + 実行時間 + オーバーヘッド
```

例：

```text
待ち時間         5秒
実行時間        10秒
オーバーヘッド   3秒

合計            18秒
```

## 試験でのポイント

> オーバーヘッドは、本来の処理以外に必要となる余分な処理や時間。
