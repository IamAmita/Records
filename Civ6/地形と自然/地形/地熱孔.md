**地熱孔（ちねつこう、Hydrothermal Vent）**とは、**海底から高温の熱水が噴き出す場所**のことである。

主に水深2,000～4,000mほどの深海にあり、海底火山やプレート境界付近で多く見られる。

## どのようにできるのか

地熱孔は次のような過程で形成される。

1. 海水が海底の割れ目から地下へしみ込む。
2. 地下深くでマグマによって数百℃まで加熱される。
3. 加熱された海水が岩石中の金属や鉱物を溶かし込む。
4. 高温の熱水が海底から勢いよく噴き出す。

噴き出した熱水は周囲の冷たい海水と混ざり、含まれていた鉱物が沈殿して煙突状の構造物（チムニー）を形成する。

## ブラックスモーカーとホワイトスモーカー

地熱孔には代表的に2つのタイプがある。

### ブラックスモーカー

熱水の温度は300～400℃に達することがあり、鉄や硫化物などの微粒子を多く含むため、黒い煙のように見える。

### ホワイトスモーカー

温度は100～300℃程度で、バリウムやカルシウム、シリカなどを多く含むため、白っぽく見える。

## 特異な生態系

地熱孔周辺では、太陽光がまったく届かないにもかかわらず、多くの生物が生息している。

その基盤となるのが**化学合成細菌**である。

これらの細菌は、熱水に含まれる硫化水素やメタンなどを利用して有機物を作る「**化学合成**」を行う。

この細菌を基礎として、

* チューブワーム
* 二枚貝
* エビ
* [蟹](../動物/蟹.md)
* タコ
* 魚類

など、多様な生物が独特の生態系を築いている。

## 科学的な重要性

地熱孔はさまざまな分野で重要な研究対象となっている。

* **生命の起源**：初期地球では地熱孔周辺で生命が誕生した可能性があるという仮説がある。
* **地球内部の研究**：プレート運動や海底火山活動を理解する手がかりとなる。
* **資源**：熱水が運ぶ金属が沈殿してできる「海底熱水鉱床」には、[銅](../../資源/銅.md)・亜鉛・金・[銀](../../資源/銀.md)などが含まれ、資源として注目されている。

## 地熱孔と間欠泉の違い

| 地熱孔                | 間欠泉              |
| ------------------ | ---------------- |
| 海底にある              | 陸上にある            |
| 常時またはほぼ連続して熱水が噴出する | 一定間隔で熱水や蒸気を噴き上げる |
| 深海の高い水圧下にある        | 大気中に噴出する         |
| 深海生物の生態系を支える       | 地熱活動の一形態として見られる  |

地熱孔は、太陽光に依存しない生態系が成立する数少ない環境であり、地球科学・生物学・資源学の観点から非常に重要な自然現象となっている。
