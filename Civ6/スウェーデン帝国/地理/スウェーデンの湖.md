# スウェーデンの湖

**スウェーデンは世界有数の「湖の国」**だ。

実際、

* 湖の数：約**10万個**（面積1ヘクタール以上）
* 小さな池まで含めると数十万個ともいわれる
* 国土の**約9％**が湖や河川

と、日本と比べても圧倒的に湖が多い。

## なぜこんなに湖が多いのか

最大の理由は**氷河**だ。

約2万年前の最終氷期には、スウェーデン全土が厚さ数千メートルにも及ぶ氷床に覆われていた。

その氷が、

1. 地面を削る
2. 深いくぼみを作る
3. 約1万年前に氷が融ける
4. くぼみに水がたまる

という過程を経て、無数の湖ができた。

日本でいう火山湖とは成り立ちが全く違い、「**氷河湖**」がほとんどだ。

## フィンランドはもっとすごい

実は、お隣のフィンランドはさらに湖が多い。

フィンランドは

* 約18万7,000の湖

があり、「**千の湖の国**」という愛称で知られる（実際には「千」どころではない）。

これはスウェーデンと同じく、氷河によって形成された地形だからだ。

## スウェーデン帝国への影響

17世紀の[スウェーデン帝国](../歴史/スウェーデン帝国.md)では、この湖の多さが国づくりにも大きな影響を与えた。

* 湖と川が天然の交通路になった。
* 森林から切り出した木材を川で運べた。
* 鉄鉱石や銅を湖・河川経由で港へ輸送できた。
* 湖や湿地が天然の防衛線としても機能した。

たとえば、[クラール川](クラール川.md) → [ヴェーネルン湖](ヴェーネルン湖.md) → ヨータ川という水系は、北海までつながる重要な物流ルートだった。

だからGoogleマップで見ると「湖が多すぎる」と感じるのは自然なことで、スウェーデンの地形そのものが**氷河によって彫刻された巨大な水と森の国**だと言える。
