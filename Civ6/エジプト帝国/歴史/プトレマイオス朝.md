# プトレマイオス朝

**プトレマイオス朝**は、アレクサンドロス大王の死後、彼の部下だったプトレマイオス1世がエジプトに築いた王朝だ。
紀元前305年ごろから紀元前30年まで、約275年間続いた。

最後の君主が[クレオパトラ7世](クレオパトラ.md)で、彼女の死によって王朝は滅亡し、エジプトはローマの支配下に入った。

## 成立

紀元前332年、マケドニア王アレクサンドロス大王が、当時アケメネス朝ペルシアの支配下にあったエジプトを征服した。

エジプト人にとってペルシア支配は必ずしも好ましいものではなかったため、アレクサンドロスは比較的歓迎された。彼はエジプトの伝統に従ってファラオとして認められ、地中海沿岸には新都市**アレクサンドリア**を建設した。

しかし紀元前323年、アレクサンドロスが明確な後継者を残さずに急死する。

その結果、巨大な帝国は部下の将軍たち、いわゆる**ディアドコイ**によって分割された。

このときエジプトを支配したのが、アレクサンドロスの将軍の一人である**プトレマイオス**だった。

プトレマイオスは最初から王を名乗ったわけではない。はじめはアレクサンドロス帝国の一部を管理する総督としてエジプトを統治していた。

やがて後継者戦争が長期化し、帝国再統一の可能性がなくなると、紀元前305年ごろに王を名乗った。

これが**プトレマイオス1世ソテル**であり、プトレマイオス朝の始まりになる。

「ソテル」はギリシア語で「救済者」という意味だ。

## ギリシア人によるエジプト王朝

プトレマイオス朝はエジプトを支配した王朝だが、王家の出自はエジプト人ではなく、**マケドニア系ギリシア人**だった。

そのため、王朝の上層部ではギリシア語が使われ、宮廷文化や政治制度にもギリシア的な要素が強かった。

特に重要なのが、次の二重構造だ。

* ギリシア人社会では、ヘレニズム世界の王として振る舞う
* エジプト人社会では、伝統的なファラオとして振る舞う

つまりプトレマイオス朝の君主は、文化によって異なる顔を使い分けていた。

ギリシア語の文書では「バシレウス」、つまりギリシア的な王として表現される。一方、エジプトの神殿壁画や碑文では、伝統的な王冠をかぶり、神々に供物を捧げるファラオとして描かれた。

これは単なる演出ではなく、支配の正統性を保つために不可欠だった。

## 首都アレクサンドリア

プトレマイオス朝の首都は、ナイル川流域の伝統的な都市ではなく、地中海沿岸の**アレクサンドリア**だった。

アレクサンドリアは、エジプトの首都であると同時に、地中海世界有数の国際都市でもあった。

住民には、

* ギリシア人
* マケドニア人
* エジプト人
* ユダヤ人
* シリア人
* その他の地中海・西アジア系住民

などがいた。

都市には巨大な港があり、エジプトの穀物やパピルス、ガラス製品、香料、象牙などが取引された。

アレクサンドリアは単なる政治都市ではなく、商業・学術・文化の中心だった。

## アレクサンドリア図書館

プトレマイオス朝を象徴する存在の一つが、**アレクサンドリア図書館**だ。

正確な成立時期や規模には不明点もあるが、プトレマイオス1世またはプトレマイオス2世の時代に整備されたと考えられている。

図書館は「ムセイオン」と呼ばれる研究機関の一部だった。

ムセイオンとは、学芸の女神ムーサたちに捧げられた施設という意味で、現代の「ミュージアム」の語源でもある。

ここには各地から学者が集められ、

* 文学
* 文献学
* 数学
* 天文学
* 医学
* 地理学
* 哲学

などの研究が行われた。

有名な学者としては、

* ユークリッド
* エラトステネス
* アリスタルコス
* ヘロフィロス
* カリマコス

などがいる。

ただし、全員が図書館に直接所属していたかは人物ごとに事情が異なる。

アレクサンドリア図書館は、当時存在した書物を可能な限り集めようとしたとされる。

後世には「港に入った船の書物をすべて写させた」という逸話もあるが、どこまで事実かは断定できない。

## ファロス島の大灯台

アレクサンドリア沖のファロス島には、巨大な灯台が建設された。

これが**アレクサンドリアの大灯台**で、古代世界の七不思議の一つに数えられる。

建設はプトレマイオス1世の時代に始まり、プトレマイオス2世の時代に完成したとされる。

高さは史料によって異なるが、100メートルを超えていた可能性がある。

この灯台は、アレクサンドリア港へ入る船を導く実用施設であると同時に、プトレマイオス朝の富と技術力を示す記念建造物でもあった。

## 宗教政策

プトレマイオス朝は、ギリシア人支配者でありながら、エジプトの伝統宗教を廃止しなかった。

むしろ神殿を建設・修復し、神官団を保護した。

現代に残るエジプト神殿の中には、意外にも古王国や新王国ではなく、プトレマイオス朝時代に建てられたものが多い。

代表的なものには、

* エドフ神殿
* デンデラ神殿
* フィラエ神殿
* コム・オンボ神殿

などがある。

建築様式は伝統的なエジプト風なので、一見すると非常に古い時代のものに見える。しかし実際には、クレオパトラに近い時代に建てられた部分も多い。

プトレマイオス朝は、エジプト神官団との協力によって統治を安定させようとした。

有名な**ロゼッタ・ストーン**も、この政治関係の中で作られた。

ロゼッタ・ストーンには、プトレマイオス5世を称える神官団の決議が、

* ヒエログリフ
* 民衆文字
* ギリシア文字

の三種類で記されている。

これが後世のヒエログリフ解読に決定的な役割を果たした。

## セラピス神

プトレマイオス朝は、ギリシア系住民とエジプト系住民を結びつけるため、宗教的な融合も進めた。

その代表が**セラピス神**だ。

セラピスは、エジプトのオシリス神や聖牛アピスと、ギリシア的な神々の特徴を組み合わせた神格だと考えられている。

外見はエジプト神のような動物頭ではなく、ひげを生やしたギリシア風の男性神として表現されることが多い。

ただし、セラピス信仰がエジプト人とギリシア人を完全に一体化したわけではない。

プトレマイオス朝社会には、文化的・法的・経済的な区別が依然として残っていた。

## 王族の兄妹婚

プトレマイオス朝で特に知られているのが、王族同士の兄妹婚だ。

王位継承を王家内部に限定し、王家の神聖性を強調するため、兄と妹、姉と弟が結婚する例が繰り返された。

この慣習を本格的に採用したのが、プトレマイオス2世と姉のアルシノエ2世だ。

エジプト神話では、オシリスとイシスが兄妹でありながら夫婦でもあった。そのため、王族の兄妹婚はエジプト的伝統によって正当化することもできた。

ただし、王家の婚姻は愛情関係というより、権力と血統の維持を目的とした政治制度に近い。

王族内部での婚姻が繰り返された結果、王位をめぐる姉弟・兄妹・親子間の争いも激しくなった。

## 最盛期

王朝の最盛期は、おおむね初期の三代とされる。

### プトレマイオス1世

王朝の基礎を築いた。

エジプト本土を確保し、アレクサンドリアを首都として整備した。また、アレクサンドロス大王の遺体をエジプトへ運び、自らの支配の正統性に利用した。

### プトレマイオス2世

経済・海軍・文化の面で王国を大きく発展させた。

姉アルシノエ2世と共同統治し、王家の神格化を進めた。アレクサンドリア図書館やムセイオンの整備も、この時代に進んだとされる。

紅海方面の交易も盛んになり、アフリカ東岸やアラビア、インド方面との交易路が発展した。

### プトレマイオス3世

軍事的には王朝の最盛期を築いた。

セレウコス朝との戦争でシリア方面へ進撃し、一時はメソポタミア方面にまで影響力を伸ばしたとされる。

この時代、プトレマイオス朝は東地中海有数の大国だった。

## セレウコス朝との対立

プトレマイオス朝の最大の競争相手は、同じくアレクサンドロス帝国の後継国家である**セレウコス朝**だった。

両国は、現在のシリア、レバノン、イスラエル、パレスチナにあたる地域をめぐって、何度も戦争を行った。

これを**シリア戦争**と呼ぶ。

争点の中心は、コイレ・シリアと呼ばれる地域だった。

この地域は、

* エジプト防衛の緩衝地帯
* 地中海東岸の港湾地帯
* 通商路の結節点
* 軍事拠点

として重要だった。

プトレマイオス朝とセレウコス朝は、婚姻同盟を結ぶこともあれば、すぐに再び戦争を始めることもあった。

クレオパトラという名前も、もともとはセレウコス朝王家からプトレマイオス朝へ嫁いだ王妃を通して広まった。

後期プトレマイオス朝の女王にクレオパトラが多いのは、そのためでもある。

## エジプト人との関係

初期のプトレマイオス朝では、軍事・行政・都市社会の主要な地位をギリシア系住民が占める傾向が強かった。

エジプト人も官僚や神官、兵士として参加したが、社会的にはギリシア系住民が優遇される場面が多かった。

エジプトには、ギリシア人都市と伝統的なエジプト人社会が並立していた。

法律も一つに完全統一されていたわけではない。

* ギリシア人にはギリシア系の法制度
* エジプト人にはエジプト系の法制度
* 国家行政には王室の法令

というように、複数の制度が重なっていた。

この分離は王朝の安定に役立つ一方、緊張の原因にもなった。

特に後期になると、重税や王位争い、地方統制の弱体化によって、上エジプトを中心に反乱が発生する。

紀元前2世紀には、一時的にエジプト人の対抗王が上エジプトを支配したこともあった。

## 経済

プトレマイオス朝の富の基盤は、ナイル川流域の農業だった。

エジプトは古代地中海世界でも特に生産力の高い穀倉地帯で、小麦や大麦を大量に生産した。

王朝は農地、灌漑、徴税を厳格に管理し、国家収入を確保した。

また、多くの産業や商品が王室の統制下に置かれた。

代表的なのは、

* 穀物
* 油
* パピルス
* 塩
* 鉱物
* 織物

などだ。

王朝は非常に発達した官僚制を持ち、土地面積、作物、家畜、人口、納税などを細かく記録した。

大量のパピルス文書が残っているため、プトレマイオス朝は古代社会の中でも経済や日常生活を詳しく研究できる時代になっている。

ただし、国家の厳格な徴税と統制は、農民にとって重い負担にもなった。

税を逃れるために村を離れる農民や、神殿へ逃げ込む者もいた。

## 軍事

プトレマイオス朝軍の中心は、当初はマケドニア式の軍隊だった。

重装歩兵が長い槍を持ち、密集隊形を組む**ファランクス**が主力となる。

王朝はギリシア・マケドニア系の兵士をエジプトへ移住させ、土地を与える代わりに軍務を負わせた。

しかし長期的には、ギリシア系兵士だけでは軍を維持できなくなり、エジプト人兵士の比重も増していった。

紀元前217年のラフィアの戦いでは、多数のエジプト人兵士がセレウコス朝との戦いに動員された。

戦争には勝利したが、武装し組織化されたエジプト人兵士が自信を持ち、その後の反乱につながったという見方もある。

また、プトレマイオス朝は強力な海軍を保有し、キプロスやエーゲ海の島々、東地中海沿岸に勢力を伸ばした。

初期には陸上帝国というより、海上拠点と港湾を結ぶ海洋勢力としての性格も強かった。

## 王朝の衰退

プトレマイオス朝の衰退には、一つではなく複数の原因がある。

### 王位継承争い

王家内部で兄弟姉妹が結婚し、同時に王位を争ったため、内戦が頻発した。

王、王妃、王太后、王子、王女、それぞれの側近集団が対立し、暗殺や追放が繰り返された。

### 財政難

対外戦争、宮廷費用、傭兵への報酬によって財政が悪化した。

税負担が増えると農村社会の不満が高まり、生産と徴税も不安定になった。

### 領土喪失

セレウコス朝やマケドニア、その他の勢力との戦争によって、海外領土を徐々に失った。

特にコイレ・シリアを失ったことは、軍事的にも経済的にも大きかった。

### 国内反乱

上エジプトでは中央政府に対する反乱が起こり、王朝の支配が一時的に及ばなくなることもあった。

### ローマへの依存

最終的に最も大きな要因となったのが、ローマの台頭だ。

プトレマイオス朝は、セレウコス朝などの脅威に対抗するため、ローマとの関係を利用した。

しかしローマの力が強まるにつれ、エジプト王の即位や王位争いまでローマの判断に左右されるようになる。

王位を取り戻すためにローマへ多額の借金をしたり、ローマの有力政治家へ賄賂を贈ったりする王も現れた。

形式上は独立王国でも、実質的にはローマの保護国に近づいていった。

## クレオパトラ7世の登場

王朝末期に現れたのが[クレオパトラ7世](クレオパトラ.md)だ。

彼女の父プトレマイオス12世の時代には、エジプトはすでにローマの強い影響下にあった。

紀元前51年、クレオパトラ7世は弟プトレマイオス13世と共同統治を始める。

しかし宮廷内で対立し、一度は追放された。

クレオパトラは、ローマ内戦の勝者となったカエサルと同盟し、王位に復帰する。

その後、カエサル暗殺後にはマルクス・アントニウスと結び、エジプトの独立と王朝の拡大を目指した。

一時的には東地中海に大きな勢力圏を築いたが、オクタウィアヌスとの対立に敗れる。

## 滅亡

紀元前31年、アクティウムの海戦でアントニウスとクレオパトラの連合軍が敗北した。

翌紀元前30年、オクタウィアヌス軍がアレクサンドリアを占領する。

アントニウスとクレオパトラは相次いで自死した。

クレオパトラとカエサルの息子とされたカエサリオン、正式にはプトレマイオス15世も処刑された。

これによってプトレマイオス朝は滅亡し、エジプトはローマ領となった。

ただし、エジプトは普通の元老院属州にはされず、皇帝が直接強く管理する特殊な地域になった。

これはエジプトが、

* ローマの重要な穀物供給地
* 莫大な税収源
* 軍事的に防衛しやすい地域
* 対抗者が利用すれば危険な富裕地帯

だったためだ。

## 主な君主

王の多くが「プトレマイオス」という同じ名前を使ったため、番号で区別される。

| 君主         | 主な特徴             |
| ---------- | ---------------- |
| プトレマイオス1世  | 王朝創始者            |
| プトレマイオス2世  | 文化・経済・海軍の発展      |
| プトレマイオス3世  | 軍事的最盛期           |
| プトレマイオス4世  | ラフィアの戦いで勝利       |
| プトレマイオス5世  | ロゼッタ・ストーンの王      |
| プトレマイオス6世  | セレウコス朝と兄弟間対立に苦しむ |
| プトレマイオス8世  | 長期の王家内紛          |
| プトレマイオス12世 | ローマへの依存を強めた      |
| プトレマイオス13世 | クレオパトラ7世と対立      |
| プトレマイオス14世 | クレオパトラ7世との共同統治者  |
| プトレマイオス15世 | カエサリオン、最後の名目的国王  |

ただし王の番号は、古代当時に使われていたものではなく、近代の研究者が整理のために付けたものだ。

王妃や女王には「クレオパトラ」「ベレニケ」「アルシノエ」という名前が繰り返し使われた。

このため、王朝史は同名人物が非常に多く、かなり分かりにくい。

## 古代エジプト史の中での位置づけ

プトレマイオス朝は、古王国や新王国とはかなり性格が異なる。

新王国はエジプト人王朝がシリアやヌビアへ勢力を伸ばした時代だった。

一方、プトレマイオス朝は、ギリシア系王家がエジプトの伝統を利用しながら統治した**ヘレニズム国家**だった。

そのため、プトレマイオス朝時代のエジプトは、

> エジプト文明がギリシア文明に置き換えられた社会

ではない。

むしろ、

> エジプト文化とギリシア文化が並存し、一部で融合しながらも、完全には一体化しなかった社会

と考える方が正確だ。

## 要するに

プトレマイオス朝は、

* アレクサンドロス大王の後継者が建てた
* マケドニア系ギリシア人の王朝
* エジプトの伝統的ファラオ制度を利用した
* アレクサンドリアを中心に繁栄した
* 学問・交易・海軍力で大きな影響を与えた
* 後期には内紛とローマ依存によって弱体化した
* クレオパトラ7世の死で滅亡した

王朝だ。

古代エジプト文明の末期であると同時に、ギリシア世界とローマ世界をつなぐ重要な時代でもある。
