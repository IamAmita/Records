# 張掖丹霞

[![China's Rainbow Mountains](https://images.openai.com/static-rsc-4/UWwy0g4olE7y1joC2b0owOpnaUe8LflK5COCqlU8fWaa3_rwdrbl5CMPE7wMrURu8dJS3RfE0xHx6vxeKWKq_t46tEthXd8O6ZYO_8ucLzRfb3hZ8gnzaGD5enPTfrpWBBdOU3uWPE7BEcQiSDFT4w8QAzFenjk5xFE7iWipOr4?purpose=inline)](https://www.catalystplanet.com/travel-and-social-action-stories/chinas-rainbow-mountains?utm_source=chatgpt.com)

**張掖丹霞（ちょうえきたんか）**は、中国北西部の**甘粛省張掖市**に広がる、赤・黄・橙・褐色などの縞模様を持つ奇岩・丘陵地帯だ。英語では**Zhangye Danxia**、鮮やかな外観から「レインボーマウンテン」と呼ばれることもある。

## どこにあるか

張掖市は、中国北西部を東西に伸びる**[河西回廊](河西回廊.md)**の中部に位置している。

河西回廊は、

> 黄河流域 → 武威 → 張掖 → 酒泉 → 敦煌 → 西域

を結ぶ、古代シルクロードの重要な交通路だった。

張掖丹霞の主要な観光区域は、張掖市街から西方にあり、祁連山脈北麓の乾燥地帯に広がっている。現在の張掖ユネスコ世界ジオパークは、粛南ユグル族自治県を中心に総面積約1,290平方キロメートルに及ぶ。

## 「丹霞」とは

**丹霞地形**とは、主として赤色の砂岩や礫岩が、地殻変動と侵食を受けて形成された地形を指す中国由来の地質用語だ。

「丹」は赤、「霞」は夕焼けや朝焼けを意味するため、文字どおりには、

> 赤い霞のように見える地形

という意味になる。

丹霞地形は張掖だけにあるわけではなく、中国南部を中心に各地で見られる。ただし張掖丹霞は、一般的な丹霞地形よりも**色彩豊かな地層**が広く露出していることが特徴だ。

## なぜ虹色に見えるのか

山が人工的に塗られているわけではない。

長い時間をかけて堆積した砂岩・泥岩・礫岩などの層に、鉄をはじめとする鉱物が含まれているため、地層ごとに異なる色が現れる。

主な色の違いは、おおむね次のように説明できる。

| 色         | 主な要因            |
| --------- | --------------- |
| 赤・赤褐色     | 酸化鉄を多く含む地層      |
| 黄・黄褐色     | 鉄鉱物の状態や砂岩・泥岩の組成 |
| 灰色・白色     | 石灰質・粘土質成分など     |
| 暗褐色・黒っぽい色 | 有機物や異なる鉱物組成     |
| 緑がかった色    | 還元状態の鉄鉱物や粘土鉱物など |

ただし、色を単純に「この鉱物なら必ずこの色」と一対一で対応させるのは正確ではない。岩石の粒子、鉱物組成、酸化状態、風化の程度、水分、光の当たり方などが複合的に影響している。

## どうやって現在の形になったのか

形成過程は、大まかに四段階で説明できる。

### 1. 地層が堆積する

かつてこの地域には、河川・湖・扇状地などの環境があり、砂、泥、礫などが繰り返し堆積した。

堆積環境や含まれる鉱物が時期ごとに異なったため、色の異なる地層が重なった。

### 2. 地層が固まる

堆積物が長い年月をかけて圧縮され、砂岩、泥岩、礫岩などになった。

### 3. 地殻変動で持ち上がる

インドプレートとユーラシアプレートの衝突に関係する地殻変動によって、祁連山脈周辺の地層が隆起し、傾き、褶曲した。

もともとほぼ水平だった色付きの層が、斜めや波状に並んで見えるのはこのためだ。

### 4. 風雨に削られる

乾燥した気候の中で、風、雨、寒暖差、凍結と融解などによる侵食が進んだ。

柔らかい地層が速く削られ、硬い地層が残ることで、

* なだらかな丘
* 深い溝
* 尖った尾根
* 波打つ斜面
* 柱状・城壁状の岩体

が形成された。

つまり張掖丹霞は、

> 色の異なる地層が堆積し、地殻変動で傾き、侵食によって内部の縞模様が露出した景観

だ。

## 七彩丹霞と氷溝丹霞

「張掖丹霞」と呼ばれる範囲には、性格の異なる複数の景観区域がある。

### 七彩丹霞

観光写真で最もよく見る、赤・橙・黄・白などの縞模様が連なる区域。

中国語では**七彩丹霞**と呼ばれ、直訳すると「七色の丹霞」だ。

特徴は、

* 色彩の鮮やかさ
* 波状の地層
* なだらかな丘陵
* 展望台からの広い眺望

にある。

一般に「中国のレインボーマウンテン」と呼ばれる写真の多くは、こちらで撮影されている。

### 氷溝丹霞

こちらは色彩よりも、侵食によって生まれた岩峰や断崖の形が中心となる区域だ。

* 城壁
* 塔
* 宮殿
* 柱
* 動物

のように見える岩体が多く、雄大で険しい景観を持つ。

簡単に分けると、

> 七彩丹霞＝色を見る場所
> 氷溝丹霞＝形を見る場所

という違いになる。

## 写真と実物の色

張掖丹霞の写真には、色彩が極端に鮮やかなものがある。

実際に色付きの地層が存在することは間違いないが、写真によっては、

* 彩度の強調
* コントラストの調整
* HDR加工
* 特定の時間帯の光
* 雨上がりによる岩肌の湿り

などによって、肉眼より強く発色して見えることがある。

特に、真昼の強い光よりも朝夕の斜光の方が凹凸と色の差がはっきりする。雨の後には岩肌の色が濃く見えるため、鮮烈な景観になりやすい。

したがって、「写真は全部偽物」というわけではないが、すべての写真が普段の肉眼に近いとも限らない。

## シルクロードとの関係

張掖そのものは、古代シルクロードの河西回廊における重要都市だった。

漢の武帝の時代、前漢が匈奴勢力を河西回廊から退けた後、張掖郡が置かれた。「張掖」という名称は、一般に「国の腕を張り、西域へ伸ばす」という政治的意味に関連づけられている。現在も張掖は河西回廊中部の歴史都市として位置づけられている。

ただし、張掖丹霞の地形そのものがシルクロードの主要宿場だった、という意味ではない。

重要なのは、

* 張掖市がシルクロード都市だった
* その周辺に丹霞地形がある
* 現代ではシルクロード観光と組み合わせて訪問される

という関係だ。

## 世界遺産なのか

ここは少し混同されやすい。

中国には「**中国丹霞**」という名称でユネスコ世界遺産に登録された複数の丹霞地形がある。しかし、その世界遺産は中国南部の6地域によって構成されており、張掖丹霞はその構成資産ではない。

一方、張掖は**ユネスコ世界ジオパーク**として認定されている。世界遺産と世界ジオパークは別の制度だ。張掖ユネスコ世界ジオパークは、色彩豊かな丘陵だけでなく、祁連山北麓の地質構造や地域文化も含む広い範囲を対象としている。

整理すると、

* **張掖丹霞＝世界遺産ではない**
* **張掖地域＝ユネスコ世界ジオパーク**
* **「中国丹霞」という別の世界遺産が存在する**

ということだ。

## 観光地としての特徴

七彩丹霞景区では、地形保護のため、観光客が自由に斜面を歩き回る形式ではない。

通常は、

* 景区内の専用バスで移動
* 複数の展望台で下車
* 遊歩道や展望施設から観察
* 再びバスで次の地点へ移動

という見学方法になる。

岩肌は柔らかく、踏み荒らすと長期間跡が残る。そのため、指定された道や展望台から外れないことが重要だ。

景観を見るなら、一般には夕方の斜光が適している。地元観光情報でも秋や夕景が特に美しい時期として紹介されているが、営業条件や入場制度は変わる可能性がある。

## 気候

張掖周辺は乾燥した大陸性気候だ。

* 夏は日差しが強い
* 冬は非常に寒い
* 昼夜の寒暖差が大きい
* 雨は比較的少ない
* 日陰が少ない
* 風が強いことがある

この乾燥した気候は植物による地表の被覆を少なくし、色の付いた岩層を見えやすくしている。一方で、まれな強い雨が侵食を進める重要な要因にもなる。

## 要するに

張掖丹霞は、

> 中国甘粛省の河西回廊にある、色の異なる堆積岩層が地殻変動で傾き、風雨の侵食によって露出した地形

だ。

特に七彩丹霞では、赤・橙・黄・白などの地層が波打つように連なり、巨大な絵画のような景観を作っている。

人工的に着色された山ではなく、**堆積・隆起・褶曲・侵食という長大な地質過程が可視化された場所**と考えると分かりやすい。
