**スフィンクス**とは、**人間の頭とライオンの体を持つ神話上・象徴上の存在**、またはその姿をした像を指す。

最も有名なのは、エジプトのギザの大スフィンクスである。

## 古代エジプトのスフィンクス

古代エジプトでは、スフィンクスは**王権・神聖さ・守護**の象徴だった。

ライオンは強さや勇気の象徴であり、人間の頭は知恵や王の人格を表している。そのため、スフィンクスは

* 王の権威
* 神の加護
* 神殿や墓を守る守護者

という意味を持っていた。

### ギザの大スフィンクス

ギザの大スフィンクスは、

* 全長：約73メートル
* 高さ：約20メートル
* 石灰岩を削り出して造られた巨大な一枚岩の彫像

である。

一般には、顔は古代エジプト第4王朝のファラオ、カフラーを表していると考えられている。

建設時期は紀元前2500年頃とされ、カフラー王の[ピラミッド](ピラミッド.md)の前に建てられたと考える説が有力である。

## ギリシャ神話のスフィンクス

一方、**ギリシャ神話**のスフィンクスは、エジプトのものとは性格が異なる。

ギリシャのスフィンクスは、

* 女性の顔
* ライオンの体
* 鷲の翼
* 蛇の尾

を持つ怪物として描かれることが多い。

このスフィンクスは旅人に謎かけをし、答えられない者を食べるとされる。

有名なのが、英雄オイディプスとの逸話である。

### スフィンクスの謎

スフィンクスは次のような謎を出した。

> 「朝は四本足、昼は二本足、夕方は三本足で歩くものは何か。」

オイディプスは「**人間**」と答えた。

* 朝＝幼児（四つん這い）
* 昼＝成人（二本足）
* 夕方＝老人（杖をつくため三本足）

謎を解かれたスフィンクスは、自ら命を絶ったと伝えられている。

## エジプトとギリシャの違い

| 古代エジプト   | ギリシャ神話    |
| -------- | --------- |
| 守護・王権の象徴 | 人を試す怪物    |
| 神聖な存在    | 恐ろしい怪物    |
| 男性的な顔（王） | 女性の顔      |
| 神殿や墓を守る  | 謎かけで旅人を襲う |

同じ「スフィンクス」という名前でも、エジプトとギリシャでは役割や意味が大きく異なっている。
