**染料（せんりょう、dye）**とは、布、糸、紙、革などの素材に色を付けるために用いられる着色物質である。

顔料とは異なり、染料は素材に浸透し、化学的・物理的に結び付くことで色を定着させる。

## 染料の種類

### 天然染料

植物、動物、鉱物など自然由来の原料から作られる。

代表例

* **藍（あい）**：青色
* **茜（あかね）**：赤色
* **ウコン**：黄色
* **サフラン**：黄色～橙色
* **コチニール**（昆虫由来）：赤色
* **ログウッド**：黒色・紫色

### 合成染料

19世紀以降に化学的に製造されるようになった染料である。

特徴

* 色の種類が豊富
* 大量生産が可能
* 色あせしにくいものが多い
* 現在の染色の大部分で使用されている

## 染色の仕組み

一般的な工程は次のとおり。

1. 素材の汚れや油分を取り除く。
2. 染料を溶かした液に浸す。
3. 必要に応じて**媒染（ばいせん）**を行い、色を定着させる。
4. 洗浄・乾燥して仕上げる。

天然染料では、ミョウバンや鉄、銅などを媒染剤として用いることが多い。

## 主な用途

染料はさまざまな分野で利用されている。

* 衣類
* 糸
* 革製品
* 和紙
* 木材
* インク
* 食品（食用色素）
* 化粧品

用途によっては、耐光性や耐水性、安全性などが重視される。

## 歴史

染料の利用は数千年前から行われている。

古代エジプトやメソポタミア、中国、インドなどでは植物や昆虫から得られる天然染料で布を染めていた。

19世紀には、1856年にイギリスの化学者ウィリアム・ヘンリー・パーキンが世界初の実用的な合成染料「モーブ」を発見したことで、染色産業は大きく発展した。

## 染料と顔料の違い

| 項目     | 染料       | 顔料             |
| ------ | -------- | -------------- |
| 素材への浸透 | 浸透して定着する | 表面に付着する        |
| 水への溶解性 | 多くは溶ける   | 基本的に溶けない       |
| 主な用途   | 布、糸、革など  | 絵の具、塗料、印刷インクなど |

## 主な天然染料

| 染料    | 原料       | 色   |
| ----- | -------- | --- |
| 藍     | タデアイなど   | 青   |
| 茜     | 茜の根      | 赤   |
| ウコン   | ウコンの根茎   | 黄   |
| サフラン  | サフランの雌しべ | 黄～橙 |
| コチニール | エンジムシ    | 赤   |

---

染料は古代から現代まで、衣類や工芸品を彩る重要な材料として利用され続けており、現在では天然染料と合成染料が用途に応じて使い分けられている。
