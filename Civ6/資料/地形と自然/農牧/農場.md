**農場（のうじょう、farm）**とは、農作物の栽培や家畜の飼育を目的として利用される土地や、その経営単位のことだ。

「畑」や「田んぼ」が個々の耕作地を指すのに対し、「農場」はそれらに加えて建物や設備を含めた一つの経営全体を意味することが多い。

## 農場の構成

一般的な農場には次のようなものが含まれる。

* 耕作地（畑・水田）
* 牧草地
* 家畜舎（牛舎・豚舎・鶏舎など）
* 倉庫・納屋
* 農機具置き場
* 作業場
* 農家の住居
* 灌漑設備や用水路

## 農場の種類

**穀物農場**

* 小麦、トウモロコシ、米、大麦などを栽培する。

**果樹農場**

* リンゴ、オレンジ、ブドウなどの果樹を育てる。

**酪農場**

* 牛を飼育し、牛乳や乳製品を生産する。

**畜産農場**

* 牛、豚、羊、鶏などを飼育し、肉・卵・羊毛などを生産する。

**混合農場**

* 作物の栽培と家畜の飼育を組み合わせて経営する。

## 規模

農場の規模は地域によって大きく異なる。

* 家族経営の小規模農場
* 地域に供給する中規模農場
* 数千～数万ヘクタールに及ぶ大規模農場（企業経営など）

例えば、北米やオーストラリアには、日本の市町村に匹敵する広さを持つ農場もある。

## 農園との違い

| 用語 | 主な意味                           |
| -- | ------------------------------ |
| 農場 | 農業経営全体。作物・家畜のどちらも含む。           |
| 農園 | 主に作物の栽培を行う場所。果樹園や茶園などを指すことも多い。 |
| 畑  | 野菜や穀物などを育てる一区画の土地。             |
| 水田 | 稲を栽培するための耕地。                   |
| [牧場](牧草地.md) | 家畜を放牧・飼育する場所。                  |

---

農場は単なる耕作地ではなく、建物や設備を含めた農業経営の単位であり、[畜産](畜産.md)や耕種農業を支える基盤として地域経済に組み込まれている。
