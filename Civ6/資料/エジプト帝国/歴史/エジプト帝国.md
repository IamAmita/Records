# エジプト帝国

「**エジプト帝国**」という言葉は、一般には古代エジプトが国外へ領土を広げ、帝国的な支配を行った時代を指す呼び方だ。ただし、歴史学上の正式な王朝名として「エジプト帝国」という国家が存在したわけではない。

多くの場合、次のどちらかを意味する。

## 1. 古代エジプト新王国

最も一般的なのは、紀元前1550年ごろから紀元前1070年ごろまでの**新王国時代**を指す用法。

第18・第19・第20王朝にあたり、古代エジプトが軍事的・政治的に最も強大だった時代だ。

代表的なファラオには、

* ハトシェプスト
* トトメス3世
* アメンホテプ3世
* アクエンアテン
* ツタンカーメン
* セティ1世
* ラメセス2世

などがいる。

特にトトメス3世の時代、エジプトはシリア・パレスチナ方面へ遠征を繰り返し、現在のスーダンにあたるヌビアから、レバノン・シリア南部付近まで影響力を広げた。このため、トトメス3世は「エジプトのナポレオン」と呼ばれることもある。

ただし、その支配方法は現代的な意味での完全な領土併合とは限らない。現地の都市国家や王を存続させながら、

* エジプトへの貢納
* ファラオへの服従
* 王族の人質
* エジプト軍の駐留
* 親エジプト政権の維持

などを通して間接的に支配する場合も多かった。

したがって、新王国は「広大な領土を一つの行政制度で統治する帝国」というより、**エジプト本土を中心に、属国・朝貢国・軍事拠点を結びつけた覇権国家**に近い。

## 2. 古代エジプト文明全体の俗称

創作作品、ゲーム、一般向けの文章などでは、古王国・中王国・新王国を含めた古代エジプト国家全体を、漠然と「エジプト帝国」と呼ぶこともある。

しかし、これはかなり大ざっぱな表現だ。

古代エジプト史は約3000年にわたり、統一と分裂、外国支配を何度も繰り返している。そのため、実際には以下のように区分する方が正確だ。

| 時代      |        おおよその年代 | 特徴          |
| ------- | -------------: | ----------- |
| 初期王朝時代  | 紀元前3100～2686年頃 | 上下エジプトの統一   |
| 古王国     | 紀元前2686～2181年頃 | [ピラミッド](../../建造物/ピラミッド.md)建設の最盛期 |
| 第一中間期   | 紀元前2181～2055年頃 | 王権の分裂       |
| 中王国     | 紀元前2055～1650年頃 | 国家再統一と南方進出  |
| 第二中間期   | 紀元前1650～1550年頃 | ヒクソスなどの支配   |
| 新王国     | 紀元前1550～1070年頃 | 帝国的拡張の最盛期   |
| 第三中間期以降 |     紀元前1070年頃～ | 分裂と外国勢力の進出  |

## エジプト帝国の支配領域

新王国最盛期のエジプトは、大きく三つの地域を支配・統制していた。

**エジプト本土**
ナイル川下流のデルタ地帯と、ナイル川上流の細長い河谷地帯。国家の中心であり、ファラオと官僚機構が直接統治した。

**ヌビア**
現在のスーダン北部。金、[象牙](../../資源/象牙.md)、黒檀などの資源が豊富で、エジプトにとって非常に重要だった。「[クシュ総督](../../制度/統治・行政/総督.md)」と呼ばれる高官を置き、比較的直接的に統治した。

**シリア・パレスチナ**
現在のイスラエル、パレスチナ、レバノン、ヨルダン、シリア南部周辺。多くの都市国家を属国化し、貢納と軍事的服従を求めた。

## なぜ帝国化したのか

新王国以前、エジプトは西アジア系とされるヒクソス勢力によってナイル川デルタ地帯を支配された。

ヒクソスを追放して成立した新王国は、外国勢力の再侵入を防ぐため、国境の外側へ軍事的な緩衝地帯を築こうとした。つまり、単純な征服欲だけではなく、

> エジプトを守るためには、エジプト国外を支配しなければならない

という安全保障上の発想が帝国化の背景にあった。

そこへ、金・木材・鉱物・奴隷・軍馬・戦車などを獲得する経済的動機と、ファラオが世界秩序を維持するという宗教的理念が加わった。

## 衰退

紀元前13世紀から12世紀ごろになると、エジプトはヒッタイトなどの大国との対立、海の民の襲来、国内経済の悪化、神官勢力の拡大、王位継承の混乱に苦しむようになる。

ラメセス3世の時代には外敵を退けたものの、国家財政は疲弊した。その後、中央政府の統制力が弱まり、上エジプトではアメン神官団、下エジプトでは別の王朝が実権を握るようになった。

こうして紀元前11世紀ごろ、新王国の帝国的支配は終わった。

要するに「エジプト帝国」とは、厳密には正式国名ではなく、主として**新王国時代の、ヌビアとシリア・パレスチナに勢力を広げた古代エジプト**を表す便宜的な呼び方だ。
