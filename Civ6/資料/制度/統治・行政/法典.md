# 法典

**法典（ほうてん）**とは、**法律を体系的にまとめた文書や書物**のことだ。

簡単に言えば、**国や社会で守るべきルールを整理して書きまとめたもの**だ。

## なぜ法典が必要なのか

法律が口伝えだけだと、

* 人によって解釈が違う
* 裁判の結果がばらばらになる
* 権力者が勝手な判断をしやすい

という問題が起こる。

そこで法律を文字に書き残し、誰でも内容を確認できるようにしたのが法典だ。

## 法典の例

### ハンムラビ法典

紀元前18世紀頃、古代バビロニアの王ハンムラビが制定した法典。[イルクム](イルクム.md)制度に関する規定も含まれている。

「目には目を、歯には歯を」という言葉で有名だが、実際には商取引や結婚、相続など多くの法律も含まれている。

### 十二表法

古代ローマで市民の権利を守るために作られた法典。

ローマ法の基礎となった。

### ナポレオン法典

1804年に制定されたフランスの民法典。

近代的な民法の基礎となり、多くの国の法律に影響を与えた。

## 法律と法典の違い

* **法律**：個々のルール。
* **法典**：多くの法律を体系的にまとめたもの。

例えば、現代日本の**民法**は、契約や相続、結婚などに関する法律を一冊にまとめた法典といえる。

## 法典の役割

法典には次のような役割がある。

* 法律を分かりやすく整理する。
* 裁判の基準を統一する。
* 国民が自分の権利や義務を知ることができる。
* 権力者の恣意的な判断を抑える。

## 世界史で重要な法典

世界史では、次の法典が特によく登場する。

| 法典       | 時代       | 特徴              |
| -------- | -------- | --------------- |
| ウル・ナンム法典 | 紀元前21世紀頃 | 現存する最古級の法典。     |
| ハンムラビ法典  | 紀元前18世紀頃 | 石柱に刻まれたことで有名。   |
| 十二表法     | 紀元前5世紀   | ローマ法の出発点。       |
| ナポレオン法典  | 1804年    | 近代民法に大きな影響を与えた。 |

## 法典と憲法の違い

* **憲法**：国の基本的なルールを定める最高法規。
* **法典**：特定の分野の法律を体系的にまとめたもの（民法典、刑法典など）。

つまり、**法典とは「法律を分かりやすく整理・体系化してまとめた文書」**のことで、古代から現代まで、社会を公平に運営するための重要な役割を果たしてきた。
