# 首長制

**首長制（しゅちょうせい）**とは、**部族・村・地域・国家などを、一人の首長（リーダー）が中心となって治める政治の仕組み**のことだ。

「首長」とは、その集団の代表者や指導者を意味する。

## 首長制の特徴

首長制では、

* 一人の首長が集団をまとめる。
* 首長が政治や裁判、祭祀(宗教儀式)を担うことが多い。
* 首長の権力は、血縁や実力、信望などによって決まる。

国家が成立する前の社会や、小規模な共同体でよく見られた。

## 首長はどのように選ばれるか

地域や時代によって異なるが、主に次のような方法があった。

* **世襲**（親から子へ受け継ぐ）
* **有力者による選出**
* **武力や功績による就任**
* **宗教的権威による承認**

## 首長制と国家の違い

| 首長制             | 国家              |
| --------------- | --------------- |
| 比較的小規模な集団       | 広い領域を支配する       |
| 首長が直接統治する       | 官僚や役人による行政組織がある |
| 法律や制度は未発達なことが多い | 法律や行政制度が整備されている |

首長制は、国家が成立する前の段階に見られることが多い。

## 世界史での首長制

### 古代

農耕が始まると村が大きくなり、有力者が首長となって集団を率いるようになった。

やがて人口や富が増えると、首長制から王国へ発展した地域も多い。

### 日本

弥生時代には、多くの**クニ**（小さな地域国家）が存在し、それぞれに首長がいた。

その後、有力な首長が他のクニを従え、やがてヤマト政権へと発展していった。

### アフリカ・オセアニア

近代以前まで、多くの地域で首長制を採る社会が存在していた。

## 首長制と王政の違い

| 首長制        | 王政                 |
| ---------- | ------------------ |
| 小規模な共同体の支配 | 国家全体の支配            |
| 首長が中心      | 王・国王が中心            |
| 制度は比較的簡単   | 官僚制度や軍隊が整備されることが多い |

首長制が発展して、[神王](神王.md)を頂点とする王国になる場合もあった。

## 世界史で重要な理由

人類の歴史をたどると、

**狩猟採集社会**
↓
**農耕の開始**
↓
**首長制社会**
↓
**王国・国家の成立**

という流れをたどった地域が多い。

そのため首長制は、**国家が誕生する前段階の政治体制**として重要視されている。

---

つまり**首長制**とは、**一人の首長が部族や村、地域社会をまとめて統治する政治の仕組み**のことで、多くの文明で国家が成立する前に見られた政治体制だ。
