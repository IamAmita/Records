# 寡頭制

**寡頭制（かとうせい、Oligarchy）**とは、少数の人々が政治権力を握って国家を支配する政治体制をいう。

「寡頭」とは「少数の支配者」という意味であり、支配者が一人ではなく、限られた集団である点が[独裁政治](独裁政治.md)との違いである。

## 特徴

**少数の支配者による統治**

貴族、富裕層、軍人、有力な政治家、企業家など、ごく一部の人々が政治を主導する。

**一般国民の政治参加が限定される**

国民全体が政治に参加できず、一部の階層だけが政治的な影響力を持つ場合が多い。

**権力が固定化しやすい**

支配層が互いに協力して権力を維持し、新たな勢力が政治に参加しにくいことがある。

## 独裁政治との違い

| 寡頭制               | 独裁政治              |
| ----------------- | ----------------- |
| 少数の支配者が統治する       | 一人またはごく少数が強い権力を持つ |
| 権力が複数人に分散している     | 権力が一人に集中しやすい      |
| 合議や集団で意思決定することがある | 指導者の意思が大きく反映される   |

ただし、実際には両者の境界は明確ではなく、少数の支配層の中でも一人の指導者に権力が集中すると、[独裁政治](独裁政治.md)の性格を強く持つようになることもある。

## 民主政治との違い

| 寡頭制               | 民主政治                |
| ----------------- | ------------------- |
| 少数者が政治を主導する       | 国民全体が政治に参加する仕組みを持つ  |
| 権力者が限定される         | 選挙などを通じて政権交代が可能     |
| 支配層の利益が優先されることがある | 幅広い国民の利益を反映することを目指す |

## 歴史上の例

寡頭制は古代から現代までさまざまな形で見られる。

* 古代ギリシャの一部の都市国家では、有力な貴族が政治を支配する寡頭制が採用された。
* 中世から近世にかけては、一部の貴族や有力商人が政治を主導する都市国家が存在した。
* 現代でも、法律上は民主制であっても、実質的に少数の政治・経済エリートが大きな影響力を持つ状態を「寡頭制的」と表現することがある。

なお、[政治哲学](政治哲学.md)では、アリストテレスが政治体制を分類する中で、「少数者による公益のための統治」を**貴族制**、「少数者が私的利益のために支配する形」を**寡頭制**として区別した。この区別は現在でも政治思想史で重要な考え方の一つとなっている。
