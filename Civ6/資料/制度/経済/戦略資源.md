# 戦略資源

**戦略資源（せんりゃくしげん）**とは、**国家の安全保障や経済、軍事にとって特に重要で、確保することが国の存続や発展に直結する資源**のことだ。

戦争だけでなく、平時の産業やエネルギー供給にも欠かせないため、各国は安定的な確保を重視している。

## なぜ「戦略」資源なのか

もし戦略資源が不足すると、

* 軍隊を動かせない
* 工場が止まる
* 電力を供給できない
* 経済活動が大きく落ち込む

など、国全体に深刻な影響が及ぶ。

そのため、多くの国は国内で生産したり、輸入先を分散したり、備蓄したりしている。

## 主な戦略資源

### 石油

現在最も代表的な戦略資源。

* 自動車・航空機・船舶の燃料
* 発電
* 化学製品やプラスチックの原料

などに使われる。

### 天然ガス

* 発電
* 暖房
* 工業用燃料

として重要。

### 石炭

* 火力発電
* 製鉄

に欠かせない。

### 鉄鉱石

鉄や鋼を作る原料であり、

* 建物
* 橋
* 船
* 戦車

などに利用される。

### レアアース（希土類）

スマートフォンや電気自動車、風力発電設備、ミサイルなどの高性能機器に必要。

### リチウム・コバルト・ニッケル

電気自動車やスマートフォンに使われるリチウムイオン電池の材料として重要。

## 世界史との関わり

戦略資源は歴史にも大きな影響を与えてきた。

* **鉄**は古代国家の武器や農具の発展を支えた。
* **石炭**は18～19世紀の産業革命を支えた。
* **石油**は20世紀以降の工業化や軍事力の中心となり、多くの国際紛争や外交の要因となった。
* 現代では、半導体や電池に必要なレアアースやリチウムなどの重要性が高まっている。

古代においても、[銀](../../資源/銀.md)や[銅](../../資源/銅.md)は貨幣や武器の原料として国家の力を左右する戦略資源だった。これらを得るための[採鉱](../../地形と自然/採掘/採鉱.md)技術も、国家の重要な基盤だった。

## 戦略資源と普通の資源の違い

| 普通の資源               | 戦略資源               |
| ------------------- | ------------------ |
| 生活や産業に利用される資源       | 国家の安全保障や経済に特に重要な資源 |
| 代替できる場合もある          | 代替が難しいものが多い        |
| 不足しても影響が比較的小さいことがある | 不足すると国全体に大きな影響を与える |

---

つまり**戦略資源**とは、**国家の安全保障・経済・産業・軍事を支えるために特に重要な資源**のことだ。時代によって重要な資源は変わるが、現在では石油、天然ガス、レアアース、リチウムなどが代表的な戦略資源とされている。
