# 機動戦

**機動戦（きどうせん）**とは、**素早い移動力を活かして敵を翻弄し、有利な場所やタイミングで戦う戦術・戦い方**のことである。

正面から力比べをするのではなく、部隊の機動力を利用して敵の弱点を突いたり、包囲したり、補給線を断ったりすることを重視する。[孫子](../../偉人/孫子.md)が説く「勝ちやすい状況を作る」思想とも通じる考え方だ。

## 機動戦の特徴

機動戦では、次のような考え方が重要となる。

* 素早く移動する
* 敵の弱点を攻撃する
* 正面からの消耗戦を避ける
* 包囲や側面・背後への攻撃を狙う
* 主導権を握り続ける

敵を「力で押し切る」のではなく、「動きで上回る」ことが基本となる。

## 消耗戦との違い

| 機動戦        | 消耗戦         |
| ---------- | ----------- |
| 機動力を重視     | 火力・兵力を重視    |
| 敵の弱点を狙う    | 正面から戦うことが多い |
| 短期間で決着を目指す | 長期間に及ぶことが多い |
| 包囲や突破を重視   | 持久力や補給が重要   |

## 歴史上の例

### ハンニバル

第二次ポエニ戦争では、機動力を活かしてローマ軍を翻弄した。

特にカンナエの戦いでは、敵軍を包囲して大勝利を収めたことで知られる。

### モンゴル帝国

チンギス・カン率いる騎馬軍団は、高い機動力を活かして広大な地域を短期間で征服した。

[騎兵](../../地形と自然/農牧/馬.md)による素早い移動、偵察、偽装撤退などを組み合わせた戦法は、機動戦の代表例とされる。[斥候](../../ユニット/斥候.md)による情報収集も、こうした機動的な戦い方を支える重要な要素だった。

### 第二次世界大戦

20世紀になると、戦車・自動車・航空機・無線通信の発達によって機動戦は大きく発展した。

特にドイツ軍の**電撃戦（Blitzkrieg）**は、

* 戦車部隊
* 歩兵
* 航空機

を連携させて敵戦線を突破し、後方へ一気に進出する戦法で、機動戦の代表的な例として知られている。

## 現代の機動戦

現代では、

* 戦車
* 装甲車
* ヘリコプター
* 戦闘機
* ドローン
* 高度な通信システム

などを組み合わせ、状況に応じて迅速に部隊を展開することが重視されている。

ただし、都市部や山岳地帯では機動力を十分に発揮しにくく、地形や敵の防御態勢によっては消耗戦になりやすいこともある。

## 歴史的な意義

機動戦は、「敵より速く動き、敵の弱点を突く」という考え方に基づく戦い方であり、古代の騎兵戦から現代の機甲戦まで、多くの時代で重要な軍事思想となってきた。現代の軍事学でも、火力・情報・補給と並んで「機動」は戦闘力を構成する重要な要素の一つと考えられている。
