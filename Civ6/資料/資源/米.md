# 米

**米（こめ）**とは、**イネ（Oryza sativa）の種子**だ。

私たちが普段食べている米は、植物学的には**果実（[穎果](穎果.md)〈えいか〉）**であり、一般には「米粒」と呼ばれている。

## 米ができるまで

イネは一年草で、春に植えられ、秋に収穫される。

```text
イネ
 ↓
花が咲く
 ↓
受粉・受精
 ↓
種子（米）が育つ
 ↓
収穫
```

穂（ほ）にはたくさんの米粒が実る。

---

## 「米」にはいろいろな呼び方がある

収穫後の加工段階によって名前が変わる。

| 名称     | 状態             |
| ------ | -------------- |
| 籾（もみ）  | 外側の籾殻が付いたまま    |
| 玄米     | 籾殻を取り除いた状態     |
| 精米（白米） | 糠（ぬか）や胚芽を削った状態 |
| ご飯     | 炊いた白米や玄米       |

つまり、普段食べている白米は、収穫したままの姿ではない。

---

## 米粒の構造

米粒は大きく次の部分からできている。

```text
籾（収穫直後）
┌───────────┐
│ 籾殻        │
│ ┌───────┐ │
│ │ 糠層    │ │
│ │ 胚芽    │ │
│ │ 胚乳    │ │
│ └───────┘ │
└───────────┘
```

* **籾殻**：硬い外殻。収穫直後についている。
* **糠層**：ビタミンや食物繊維が豊富。
* **胚芽**：新しいイネになる部分。
* **胚乳**：デンプンが多く、私たちが主に食べる部分。

---

## なぜ主食になったのか

米にはいくつもの利点がある。

* デンプンが豊富でエネルギー源になる。
* 水田では単位面積あたりの収量が高い。
* 乾燥させれば長期間保存できる。
* 炊くだけで食べられる。

そのため、東アジアや東南アジアでは数千年前から重要な主食となっている。

---

## ジャポニカ米とインディカ米

世界で栽培される米は大きく2種類に分けられる。

### ジャポニカ米

* 日本、中国東北部、韓国など
* 粒が丸く短い
* 粘りが強い
* 日本のご飯はこちら

### インディカ米

* インド、タイ、ベトナムなど
* 粒が細長い
* 粘りが少ない
* チャーハンやカレー、ビリヤニなどに向いている

## 稲作と水田

イネは水が大好きな植物というイメージがあるが、実は**水中植物ではない**。

水田で育てる主な理由は、

* 雑草を抑える。
* 水を安定的に供給する。
* 地温を調節する。

ためだ。

イネは根に酸素を送る特殊な組織（通気組織）を持っているため、水を張った田んぼでも育つことができる。一方、多くの雑草は水中では育ちにくいため、イネが有利になるのだ。

この性質を利用した水田農業は、東アジアの人口増加と文明の発展を支える重要な技術となった。
