# クラール川

[![Klarälven - Vildmark i Värmland](https://images.openai.com/static-rsc-4/ORwrx6QH5Ry4Q-Rhe6qmPSDxhXUtg7kkoSvCWul5Ro5F-eDmAYpQCcwEWM2Ei_Cuc2RZr3cT3JhFxbzYfjV0yez9QqL-dro6yxg4_8N9PCeA8NMFMHdlmi0N7bQZJMEwsy5K5SRLcwobip2TtaN3JqL5w8EBpiqtRypfcDAziOY?purpose=inline)](https://www.vildmark.se/en/experiences/klaralven/?utm_source=chatgpt.com)

**クラール川**（スウェーデン語：**Klarälven**）は、ノルウェーからスウェーデン中部へ流れ、最終的に**ヴェーネルン湖**へ注ぐ川だ。スウェーデン語名は「澄んだ川」という意味を持つ。

## 基本情報

| 項目           | 内容           |
| ------------ | ------------ |
| 全長           | 約460km       |
| 流域国          | ノルウェー、スウェーデン |
| スウェーデンでの主な流域 | ヴェルムランド地方    |
| 河口           | ヴェーネルン湖      |
| 河口付近の都市      | カールスタード      |

上流部はノルウェー領を流れ、ノルウェーでは主に**トリシル川（Trysilelva）**と呼ばれる。スウェーデンへ再び入るとクラール川となり、ヴェルムランド地方を北から南へ縦断する。

## スウェーデン最長の水系

クラール川の水はヴェーネルン湖へ入り、そこから**ヨータ川**として西へ流れ、カテガット海峡へ注ぐ。

そのため、

**クラール川―ヴェーネルン湖―ヨータ川**

を一つの水系として考えると、全長は約720kmとなり、スウェーデンおよび北欧で最長級の河川水系になる。

## 蛇行する川

クラール川中流部は、大きく左右に曲がる**蛇行**で有名だ。

流れによって外側の岸が削られ、内側に砂や土が堆積するため、河道が少しずつ移動していく。周辺には砂州や三日月湖など、川の働きによって形成された地形が見られる。

## 林業と木材流送

クラール川流域は森林が豊富で、かつては伐採した丸太を川に浮かべて下流へ運ぶ**木材流送**が盛んだった。

17世紀頃から重要性が高まり、ヴェルムランド地方の製材業・製紙業を支えた。クラール川での大規模な木材流送は1991年まで続いたとされる。

## カールスタードとの関係

クラール川はヴェーネルン湖へ注ぐ直前で複数の流れに分かれ、河口に三角州を形成している。その三角州に発展したのが**カールスタード**だ。

カールスタードは、

* 河川交通
* 林業
* 製材・製紙業
* ヴェーネルン湖の水運

によって発展した都市であり、クラール川とは非常に深い関係がある。

## ルーレ川との違い

| クラール川        | ルーレ川         |
| ------------ | ------------ |
| スウェーデン中部を流れる | スウェーデン北部を流れる |
| ノルウェーにもまたがる  | 主にスウェーデン国内   |
| ヴェーネルン湖へ注ぐ   | ボスニア湾へ直接注ぐ   |
| 林業・木材流送で重要   | 水力発電で特に重要    |
| 蛇行が目立つ       | ダムや急流が多い     |

つまりクラール川は、**スウェーデン中部の森林地帯を貫き、ヴェーネルン湖とヨータ川へつながる重要河川**だ。
