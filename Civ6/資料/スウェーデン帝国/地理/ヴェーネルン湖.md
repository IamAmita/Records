# ヴェーネルン湖

**ヴェーネルン湖**（スウェーデン語：**Vänern**）は、**スウェーデン最大の湖**であり、**欧州連合（EU）最大の湖**でもある。ヨーロッパ全体では、ロシアのラドガ湖・オネガ湖に次いで**3番目**の大きさを誇る。

## 基本データ

| 項目     | 内容         |
| ------ | ---------- |
| 面積     | 約5,650 km² |
| 最大水深   | 約106 m     |
| 標高     | 約44 m      |
| 主な流入河川 | クラール川      |
| 流出河川   | ヨータ川       |

クラール川から流れ込んだ水は、ヴェーネルン湖を経て**ヨータ川**へ流れ、最終的にカテガット海峡（北海）へ注ぐ。

---

## 「内海」のような湖

ヴェーネルン湖は非常に広大で、対岸が見えない場所もあるほどだ。

湖には**約22,000もの島や小島、岩礁**があり、ヨーロッパ最大級の淡水群島が形成されている。その景観は海に近く、「スウェーデンの内海」とも表現される。

---

## 歴史的重要性

ヴェーネルン湖は古くからスウェーデンの交通・物流の中心だった。

* 北部の森林地帯から木材を運ぶ
* 鉄や銅などの鉱産物を輸送する
* 周辺都市の交易を支える

といった役割を担い、スウェーデン王国や[スウェーデン帝国](../歴史/スウェーデン帝国.md)の経済発展に大きく貢献した。

19世紀には**ヨータ運河**が整備され、ヴェーネルン湖と東海岸側のバルト海が結ばれたことで、国内を横断する重要な水運ルートとなった。

---

## 周辺の都市

湖岸にはいくつもの都市が発展している。

* カールスタード：クラール川の河口に位置する北岸最大の都市。
* リードショーピング：南岸の港町。
* ヴェーナシュボリ：ヨータ川の流出口にある都市。

これらの都市は、湖を利用した交易や工業によって発展してきた。

---

## 豊かな自然

ヴェーネルン湖には約38種類の魚が生息し、漁業やスポーツフィッシングが盛んである。また、砂浜や湿地、岩礁、森林など多様な自然環境が広がり、水鳥の重要な生息地にもなっている。湖内にはデューロ国立公園があり、手つかずの群島景観を楽しむことができる。

## クラール川との関係

[クラール川](クラール川.md)は、このヴェーネルン湖へ注ぐ最大の河川だ。

**クラール川 → ヴェーネルン湖 → ヨータ川 → 北海**

という一連の水系は、スウェーデンで最も重要な河川・湖沼ネットワークの一つであり、古くから交通・交易・産業を支えてきた。
