# レンジャー

**レンジャー（Ranger）**とは、本来は「森林管理人」「巡視員」「警備員」を意味する英語である。現在では、分野によってさまざまな意味で使われている。

## 軍事におけるレンジャー

軍事分野では、高度な訓練を受けた精鋭の歩兵部隊や特殊任務部隊を指す。

レンジャー部隊は、次のような任務を担当することが多い。

* 偵察・情報収集
* 奇襲攻撃
* 空挺降下
* 山岳・森林での戦闘
* 要所の確保
* 特殊作戦の支援

レンジャー部隊は厳しい選抜と訓練を経て編成され、高い機動力と戦闘能力を備えている。

## 自然保護におけるレンジャー

自然保護の分野では、国立公園や自然保護区の管理・巡視を行う職員を指す。

主な仕事には次のようなものがある。

* 自然環境の保全
* 野生動植物の保護
* 利用者への案内・指導
* 巡視や安全管理
* 環境教育

日本では、環境省の自然保護官が「レンジャー」と呼ばれることがある。

## 一般的な意味

英語の「ranger」は、「一定の地域を巡回・管理する人」という意味から派生して、さまざまな職業や組織名に使われている。

例えば、

* 森林管理員
* 公園管理員
* 国境警備隊員
* 警備・巡視担当者

などを指す場合がある。

## コマンドーとの違い

軍事分野では「レンジャー」と「[コマンドー部隊](コマンドー部隊.md)」は似た意味で使われることもあるが、厳密には異なる概念である。

| レンジャー                   | コマンドー                         |
| ----------------------- | ----------------------------- |
| 精鋭の軽歩兵・特殊任務部隊を指すことが多い   | 奇襲・破壊工作などを専門とする特殊部隊の呼称として始まった |
| 偵察、空挺、陣地確保など幅広い任務を担う    | 奇襲、襲撃、人質救出などの特殊作戦を主な任務とする     |
| 国によっては特殊部隊の一種として位置づけられる | 現在では特殊部隊全般を指すこともある            |

---

「レンジャー」は巡視・警備・管理を行う人という本来の意味を持つ言葉であり、現在では自然保護の職員や高度な訓練を受けた軍の精鋭部隊など、文脈に応じてさまざまな意味で用いられている。
