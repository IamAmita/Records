**辺境守備隊（へんきょうしゅびたい）**とは、**国家や領土の国境・辺境地域を守るために配置された軍隊**のことである。

敵国の侵攻や遊牧民の襲撃、盗賊の侵入を防ぐとともに、国境の監視や治安維持を担う重要な部隊である。

## 主な任務

辺境守備隊の仕事は、戦争だけではない。

* 国境の監視
* 敵の侵入の早期発見
* 砦や城壁の防衛
* 関所での人や物資の管理
* 辺境住民の保護
* 街道や交易路の安全確保

平時には警備や巡回が中心であり、有事には侵攻軍を迎え撃つ最前線となる。

## 辺境守備隊が重要な理由

古代から近世にかけて、国境には自然の障害がない場所も多く、敵勢力や遊牧民が侵入しやすかった。

そのため、辺境に常駐する部隊を置くことで、

* 侵略を抑止する
* 異変を本国へ素早く伝える
* 敵の進軍を遅らせる

といった役割を果たした。

## 歴史上の代表例

### リミタネイ

後期ローマ帝国では、国境（リーメス）の防衛を担当する常駐部隊として「リミタネイ」が置かれた。

彼らは砦に駐屯し、侵入者の監視や初動防衛を担った。

### 万里の長城の守備兵

中国の歴代王朝では、万里の長城沿いに守備隊を配置し、北方の遊牧民族の侵入を監視した。[河西回廊](../../../Domain/文明/中国/地理/河西回廊.md)の西端を守る嘉峪関のように、要衝には特に重厚な防衛拠点が築かれた。

烽火台（のろし台）を利用して異変を素早く伝える仕組みも整えられていた。

### ビザンツ帝国軍

東ローマ帝国では、国境地帯に要塞を築き、辺境守備隊を配置してササン朝や後のイスラム諸勢力などへの備えを行った。

## 装備と生活

辺境守備隊の兵士は、戦うだけでなく日常的にさまざまな任務をこなした。

* 城壁や砦の維持・修理
* 巡回警備
* 見張り
* 訓練
* 地元住民との協力

多くの場合、砦の周辺には兵士や家族、商人が暮らす集落が形成されることもあった。

## 正規軍との違い

| 辺境守備隊        | 野戦軍（機動軍）     |
| ------------ | ------------ |
| 国境に常駐する      | 必要に応じて移動する   |
| 防衛・監視が中心     | 攻勢・決戦が中心     |
| 砦や城壁を拠点とする   | 戦場へ機動展開する    |
| 長期間同じ地域を担当する | 戦況に応じて配置が変わる |

多くの国家では、辺境守備隊が侵攻を食い止め、その間に野戦軍が集結して反撃するという役割分担が行われていた。[機動戦](機動戦.md)は、こうした野戦軍側の代表的な戦い方の一つである。

## 歴史的な意義

辺境守備隊は、国家の「最前線」を支える存在であった。単に敵と戦うだけでなく、国境管理、治安維持、交易路の保護、情報収集など幅広い役割を担い、国家の安全保障を支えてきた。多くの帝国では、辺境守備隊の強さが国家の安定と領土維持を左右する重要な要素となっていた。
