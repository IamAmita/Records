# 神王

**神王（しんおう）**とは、**神として崇拝される王**、あるいは**神のような存在とされた君主**のことだ。

世界史では、王が政治だけでなく宗教的な権威も持ち、「神そのもの」または「神の子孫」と考えられた国家で使われる概念だ。

## 神王の特徴

神王には、次のような特徴がある。

* 政治の最高権力者である。
* 宗教の最高指導者でもある。
* 神そのもの、または神の代理人と考えられる。
* 国民は王に宗教的な敬意を払う。

つまり、**政治と宗教が一体になった支配者**と言える。

## 神王が存在した文明

### 古代エジプト

最も有名な例は[古代エジプト](../../../Domain/文明/エジプト帝国/歴史/エジプト帝国.md)の**ファラオ**だ。

ラムセス2世などのファラオは、生きている間は神の一柱である**ホルス**の化身、死後は**オシリス**と結び付けられるなど、神聖な存在と考えられていた。

そのため、巨大な神殿やピラミッドを建設する権威を持っていた。

### メソポタミア

メソポタミアでは、王は**神に選ばれた支配者**とされることが多く、古代エジプトほど「神そのもの」と見なされることは一般的ではなかった。

### 日本

日本の古代では、天皇は神話の中で太陽神である**天照大神**の子孫とされ、神聖な権威を持つ存在と考えられてきた。ただし、歴史的な位置づけや時代ごとの考え方は変化している。

## なぜ神王が必要だったのか

古代では、自然災害や豊作・凶作が人々の生活を大きく左右していた。

そのため、

* 神々の加護を受ける王
* 神に祈りをささげる祭司としての王

が国を治めることで、国の安定や豊穣がもたらされると信じられていた。

また、「王は神である」とすることで、王の命令に従うことが宗教的にも正当化され、国家をまとめやすくなった。

## 神王と王の違い

| 王           | 神王                    |
| ----------- | --------------------- |
| 国を治める支配者    | 神、または神に極めて近い存在とされる支配者 |
| 政治が主な役割     | 政治と宗教の両方を担う           |
| 必ずしも神聖視されない | 強く神聖視される              |

---

つまり**神王**とは、**神として、あるいは神の代理人として崇拝されながら国を統治する王**のことだ。世界史では特に古代エジプトのファラオが代表的な例として知られている。
