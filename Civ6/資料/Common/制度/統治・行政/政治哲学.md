# 政治哲学

**政治哲学**とは、「国家や政治はどうあるべきか」「権力は何のために存在するのか」「正義とは何か」といった、政治の根本原理を考える哲学の分野だ。

政治学が「実際に政治がどのように行われているか」を研究することが多いのに対し、政治哲学は「どうあるべきか（規範）」を考える点に特徴がある。

## 主なテーマ

**国家とは何か**

* なぜ国家は存在するのか。
* 国家はどこまで国民を支配してよいのか。

**正義とは何か**

* 公平な社会とはどのようなものか。
* 富や機会はどのように分配されるべきか。

**自由とは何か**

* 個人の自由はどこまで認められるべきか。
* 公共の利益とのバランスをどう取るか。

**権力の正当性**

* 政府が国民に命令する権利はどこから生まれるのか。
* 国民はいつ政府に従わなくてもよいのか。

**民主主義**

* 民主主義は最善の政治制度なのか。
* 多数決だけで物事を決めてよいのか。

## 主な思想家

**プラトン**

理想国家について論じ、『国家』を著した。

**アリストテレス**

人間は「政治的動物」であると述べ、さまざまな政治体制を比較した。

**トマス・ホッブズ**

強い国家による秩序維持を重視した。

**ジョン・ロック**

自由や財産権を重視し、近代民主主義に大きな影響を与えた。

**ジャン=ジャック・ルソー**

人民主権や社会契約を論じた。

**ジョン・ロールズ**

公正としての正義を提唱し、現代政治哲学に大きな影響を与えた。

## 政治学との違い

| 政治哲学              | 政治学                |
| ----------------- | ------------------ |
| 「どうあるべきか」を考える     | 「実際にどうなっているか」を研究する |
| 正義・自由・権利・国家の理念を扱う | 選挙・政党・政策・行政などを分析する |
| 規範的・哲学的           | 実証的・社会科学的          |

---

政治哲学は「より良い政治とは何か」を理性と論理によって探究する学問であり、国家、権力、自由、平等、正義など、人間社会の根本的な問題を扱う分野である。
