# 総督

**総督（そうとく）**とは、**君主や中央政府に代わって、ある地域や[植民地](植民地.md)を統治するために派遣された最高責任者**のことだ。

簡単に言えば、**「国王や皇帝の代理として地方を治める人」**だ。

## 総督の役割

総督は、中央政府の命令を現地で実行する責任者であり、次のような仕事を行った。

* 政治・行政を行う
* 税を徴収する
* 治安を維持する
* 軍隊を指揮する
* 裁判を監督する

つまり、その地域では国王や皇帝に次ぐ強い権限を持つことが多かった。

## 世界史での例

### 古代ペルシア帝国

アケメネス朝では、広大な領土を複数の州（サトラピー）に分け、それぞれに**総督（サトラップ）**を置いた。

総督は税の徴収や行政を担当したが、反乱を防ぐために軍や財政は別の役人が監督するなど、権力が集中しすぎない工夫もされていた。

### 古代エジプト

新王国時代の[エジプト帝国](../../../Domain/文明/エジプト帝国/歴史/エジプト帝国.md)は、支配下に置いたヌビアに「**クシュ総督**」と呼ばれる高官を派遣し、比較的直接的に統治した。

### 植民地時代

ヨーロッパ諸国は植民地に総督を派遣した。

例えば、

* [イギリス領インド](../../../Domain/文明/イギリス帝国/歴史/イギリス帝国.md)の総督
* フランス領インドシナの総督
* オランダ領東インドの総督

などが有名で、本国政府を代表して植民地を統治した。

### 日本

日本でも時代によって総督という役職が置かれた。

代表例として、1895年から1945年まで存在した**台湾総督**や、1910年から1945年まで存在した**朝鮮総督**がある。これらは日本政府を代表して現地を統治した。

### 遠征帝国における総督

[ナーディール・シャー](../../../Domain/文明/イラン/歴史/ナーディール・シャー.md)が築いた征服帝国でも、各地の総督や部族長を残して貢納や軍事的服従を求める間接統治が行われた。

## 総督と知事の違い

| 総督                      | 知事          |
| ----------------------- | ----------- |
| 国王・皇帝・中央政府の代理として広い権限を持つ | 地方自治体の長     |
| 植民地や属州を統治することが多い        | 都道府県や州を運営する |
| 軍事・外交を担うこともある           | 主に行政を担当する   |

## なぜ総督が必要だったのか

昔の帝国は領土が広く、国王や皇帝がすべての地域を直接治めることはできなかった。

そのため、信頼できる人物を総督として派遣し、

* 現地の統治
* 税の徴収
* 反乱の鎮圧
* 中央政府への報告

を任せていた。

---

つまり**総督**とは、**国王や皇帝、中央政府の代理として地方や植民地を統治する最高責任者**のことだ。広大な帝国を効率よく支配するために設けられた重要な役職であり、古代ペルシア帝国や近代の植民地支配で広く見られた。
