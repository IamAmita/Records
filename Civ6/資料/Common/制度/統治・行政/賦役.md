# 賦役

**賦役（ふえき）**とは、国家や領主に対して人民が負う**税と労働の義務**を総称した言葉だ。

「賦」と「役」はもともと別の意味を持っている。

* **賦**：物やお金で納める税
* **役**：労働によって果たす義務（労役）

つまり賦役とは、「税金を納めること」と「働いて国家や領主に奉仕すること」の両方を合わせた概念である。

## 賦と役の違い

| 項目    | 賦        | 役                |
| ----- | -------- | ---------------- |
| 納めるもの | 米・布・貨幣など | 労働               |
| 内容    | 租税       | 労役               |
| 例     | 年貢、租、調   | 道路建設、堤防工事、城の建設など |

## 日本の律令制における賦役

日本の律令国家では、成人男性を中心にさまざまな賦役が課された。

代表例は次のとおり。

* **租（そ）**：田んぼの収穫の一部（稲）を納める。
* **調（ちょう）**：地方の特産物（絹・布など）を納める。
* **庸（よう）**：本来は労働を提供する義務だが、代わりに布を納めることも多かった。
* **雑徭（ぞうよう）**：道路・橋・役所などの公共工事に従事する。
* **兵役**：一定期間、兵士として勤務する。

このように、税だけではなく公共事業や軍事にも人々が動員されていた。

## 中国での賦役

中国でも歴代王朝は賦役制度を採用していた。

例えば秦の始皇帝の時代には、

* 万里の長城
* 阿房宮
* 始皇帝陵

などの巨大建築のため、多数の農民が賦役として動員された。[兵馬俑](../../../Domain/文明/中国/歴史/兵馬俑.md)が納められた始皇帝陵も、この賦役によって築かれたものだ。

隋の煬帝の時代にも、大運河建設などで大規模な賦役が課され、民衆の負担が重くなったことが隋の滅亡の一因とされている。

## 労働徴発との違い

両者は似ているが、少し意味が異なる。

* **賦役**：制度として定められた税と労働の義務。
* **[労働徴発](労働徴発.md)**：国家が特定の目的のために人々を強制的に動員すること。

つまり、賦役は平時から存在する制度であり、労働徴発は戦争や大規模建設などの際に実施される強制動員を指すことが多い。
