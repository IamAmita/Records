# 集落

**集落**とは、人々がまとまって居住し、生活を営んでいる場所のことだ。

単に家があるだけではなく、住民同士が社会や経済活動を行う共同体として機能していることが特徴になる。

## 集落の構成

一般的な集落には、次のようなものがある。

* 住宅
* 道路
* 井戸や水場
* 農地や牧草地
* 倉庫
* 寺院・教会・神社などの宗教施設
* 市場や商店
* 公民館などの共同施設

## 規模による違い

明確な人数の基準はないが、おおよそ次のように考えられる。

| 種類    | 規模の目安     |
| ----- | --------- |
| 小集落   | 数戸～数十戸    |
| 村（村落） | 数十~数百戸    |
| 町     | 数百~数千戸以上  |
| 都市    | 数万人以上が一般的 |

「集落」はこれらをまとめた総称であり、村や町も広い意味では集落に含まれる。

## 立地による分類

集落は立地によってさまざまな形になる。

* **農村集落**：農業を中心に生活する。
* **漁村集落**：海や湖で漁業を営む。
* **牧畜集落**：家畜の放牧を中心とする。
* **鉱山集落**：鉱山の採掘を目的に形成される。
* **オアシス集落**：砂漠の水源を中心に発達する。

## 形による分類

地理学では、集落の配置でも分類される。

* **集中集落**：家屋が密集している。
* **散村**：家屋が広い範囲に点在している。
* **街村**：道路や川沿いに細長く並ぶ。
* **環状集落**：中心を囲むように家が並ぶ。

## 集落と都市の違い

* **集落**：人が住み生活する場所全般を指す広い概念。
* **[都市](都市国家(概念).md)**：人口が多く、商業・工業・行政などの機能が高度に集中した大規模な集落。

つまり、すべての都市は集落の一種だが、すべての集落が都市というわけではない。

## 人口規模の目安

おおよその発展段階として、次のような目安で考えると扱いやすい。

* 開拓地：10～50人
* 小集落：50～300人
* 村：300～2,000人
* 町：2,000～2万人
* 都市：2万人以上

もちろん、これはあくまで目安であり、時代や地域によって大きく異なる。古代や中世では人口1万人でも「大都市」と呼ばれることが珍しくなかった。
