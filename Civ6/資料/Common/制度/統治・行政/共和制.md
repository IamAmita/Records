# 共和制

**共和制（きょうわせい、Republic）**とは、国家元首が世襲の君主ではなく、国民やその代表によって選ばれる政治体制をいう。

共和制の基本的な考え方は、国家は特定の家系や王族のものではなく、国民全体の公共のもの（公共の利益のための国家）であるという点にある。

## 特徴

**世襲の君主がいない**

国王や皇帝ではなく、大統領などが国家元首となることが多い。

**国家元首は選ばれる**

国民による選挙や議会での選出など、憲法で定められた方法で選ばれる。

**法に基づく統治**

国家は憲法や法律に従って運営されることを原則とする。

## 共和制と民主制の違い

共和制と民主制は混同されやすいが、異なる概念である。

| 共和制                     | 民主制             |
| ----------------------- | --------------- |
| 国家元首のあり方を示す             | 権力の担い手を示す       |
| 君主が存在しない                | 国民が政治に参加する      |
| 民主的な共和制もあれば、そうでない共和制もある | 共和制でも君主制でも成立しうる |

つまり、

* **共和制**：「誰が国家元首になるか」の制度
* **民主制**：「誰が政治の最終的な担い手か」の制度

という違いがある。

## 君主制との違い

| 共和制        | 君主制                |
| ---------- | ------------------ |
| 国家元首は選ばれる  | 国家元首は世襲            |
| 大統領などが国家元首 | 国王・女王・皇帝・天皇などが国家元首 |
| 任期があることが多い | 終身・世襲であることが多い      |

## 共和制でも民主的とは限らない

共和制だからといって、必ずしも民主的な国家とは限らない。

自由で公正な選挙が行われる共和制国家もあれば、実質的に一党支配や[独裁政治](独裁政治.md)体制となっている共和制国家もある。

このような国では国家元首は世襲ではないものの、政治的自由や政権交代が大きく制限されている場合がある。

## 現代の例

**共和制の国**

* フランス
* ドイツ
* アメリカ合衆国
* イタリア

**君主制の国**

* 日本（立憲君主制）
* イギリス（立憲君主制）
* スウェーデン（立憲君主制）

---

共和制とは「国家元首を世襲の君主とせず、公共の国家として運営する政治体制」を指す概念であり、民主制・[独裁政治](独裁政治.md)・[寡頭制](寡頭制.md)などとは別の観点から国家を分類する制度である。
