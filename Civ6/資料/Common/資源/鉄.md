**鉄（てつ）**とは、**元素記号 Fe、原子番号26の金属元素**であり、人類が最も広く利用している金属の一つである。

丈夫で加工しやすく、地球上に豊富に存在することから、建築、機械、工具、乗り物など、現代社会のあらゆる分野で使われている。

## 基本的な性質

鉄には次のような特徴がある。

* **硬く丈夫**である。
* **磁石に引き付けられる**（強磁性）。
* **熱や電気をよく伝える**。
* **さびやすい**（酸素や水と反応して酸化する）。

純粋な鉄は比較的柔らかいため、炭素などを加えて**鋼（はがね）**として使われることが多い。

## 鉄の産出

鉄は自然界では単体ではほとんど存在せず、**鉄鉱石**として採掘される。

代表的な鉄鉱石には、

* **赤鉄鉱（ヘマタイト）**
* **磁鉄鉱（マグネタイト）**
* **褐鉄鉱（リモナイト）**

などがある。

これらを高温で精錬し、不純物を取り除くことで鉄が得られる。

## 鉄器の登場

人類はまず銅や青銅を利用したが、その後、より高温で精錬する技術を獲得すると鉄を利用できるようになった。

この時代は**鉄器時代**と呼ばれる。

鉄器は、

* 武器
* 農具
* 工具

などに広く用いられ、生産性や軍事力の向上につながった。

## 鉄と青銅の違い

| 鉄                 | 青銅          |
| ----------------- | ----------- |
| 主成分は鉄             | 主成分は銅とスズ    |
| 原料が豊富             | スズが希少な地域も多い |
| より硬く丈夫（適切に加工した場合） | 鋳造しやすい      |
| 高温での精錬が必要         | 比較的低温で製造できる |

初期の鉄器は必ずしも青銅器より優れていたわけではないが、製鉄技術が発達すると鉄が主流となった。

## 現代での利用

鉄は現在でも最も重要な工業材料の一つである。

主な用途には、

* 建物の骨組み
* 橋
* 鉄道
* 自動車
* 船舶
* 工作機械
* 家電製品

などがある。

また、鉄に炭素やクロム、ニッケルなどを加えることで、

* **鋼（鋼鉄）**
* **ステンレス鋼**

など、用途に応じたさまざまな合金が作られている。

## 人体との関わり

鉄は人体にも欠かせない元素である。

血液中の**ヘモグロビン**には鉄が含まれており、酸素を全身へ運ぶ役割を担っている。

鉄が不足すると**鉄欠乏性貧血**を引き起こすことがある。

## 歴史的な意義

鉄の利用は、人類の歴史における大きな転換点となった。豊富な資源を活用できるようになったことで、農業や建築、武器製造が大きく発展し、多くの文明や国家の成長を支えた。現代でも鉄は社会基盤を支える最も重要な素材の一つであり、「文明を支える金属」ともいえる存在である。
