# 斥候

**斥候（せっこう）**とは、**本隊より先に進み、敵や地形の情報を調べて報告する役目を担う兵士**のことだ。

現代でいう**偵察兵**や**リコン（reconnaissance）部隊**に近い役割を持つ。

## 斥候の役割

戦争では、敵の位置が分からないまま進軍すると奇襲を受ける危険がある。

そこで斥候が先行して、

* 敵がどこにいるか
* 敵の人数
* 敵の武器や陣形
* 地形や道の状況
* 川や橋が渡れるか

などを調べ、本隊へ報告する。

例えば、本隊が1000人の軍隊なら、その数キロ前方を数人から数十人の斥候が進むことがある。

## 斥候に求められる能力

斥候は戦うことよりも、**見つからずに情報を持ち帰ること**が重要だ。

そのため、

* 観察力
* 判断力
* 足の速さ
* 隠密行動
* 地図を読む能力
* サバイバル能力

などが求められる。

## 斥候と兵士の違い

| 一般の兵士       | 斥候             |
| ----------- | -------------- |
| 敵と戦うことが主な任務 | 情報収集が主な任務      |
| 集団で行動する     | 少人数で行動することが多い  |
| 正面戦闘を担当     | 敵を避けて行動することが多い |

## 歴史上の斥候

多くの時代や地域で斥候は活躍した。

* **古代ローマ**では騎兵が斥候として先行することがあった。
* **モンゴル帝国**では[騎馬](../地形と自然/農牧/馬.md)の斥候が広範囲を偵察し、敵軍の情報を集めた。
* **戦国時代の日本**では、忍者や物見(ものみ)が斥候の役割を担うことがあった。

## 現代の軍隊では

現代でも斥候の役割は存在するが、「斥候」という言葉よりも

* 偵察部隊
* 偵察兵
* 偵察隊

などと呼ばれることが多い。

また、人だけでなく、

* ドローン
* 偵察機
* 衛星

なども情報収集に使われるようになっている。

## 「斥候」の語源

「斥」は「遠くへ行く」「押しやる」、「候」は「様子をうかがう」という意味がある。

つまり**「斥候」**とは、「**前方へ出て敵や周囲の様子を探る者**」という意味だ。
