# テミストクレス

**テミストクレス**とは、**古代ギリシャの[都市国家](../制度/都市国家(概念).md)アテネの政治家・軍人**だ。

紀元前5世紀に活躍し、**ペルシア戦争でギリシャを勝利に導いた最大の功労者の一人**とされている。

---

## 基本情報

テミストクレス

* 生没年：紀元前524年頃～紀元前459年頃
* 出身：アテネ
* 役職：政治家・将軍（[ストラテゴス](../../Common/制度/軍事/ストラテゴス.md)）
* 功績：アテネ海軍を強化し、サラミスの海戦を指揮

---

## なぜ有名なのか

テミストクレスは、

> **「アテネは陸軍ではなく海軍で国を守るべきだ」**

と考えた。

当時、多くの人は陸軍を重視していたが、彼は将来ペルシア帝国との大戦が起こることを見越し、海軍の整備を進めた。

その結果、アテネは強力な海軍国家へと成長した。

---

## サラミスの海戦

サラミスの海戦は、ペルシア戦争最大の決戦だ。

紀元前480年、クセルクセス1世が率いる大軍がギリシャへ侵攻した。

ギリシャ軍は数で劣っていたが、テミストクレスは狭い海峡へ敵艦隊を誘い込んだ。

```text
広い海
━━━━━━━━━━━━

      ↓ 誘導

狭いサラミス海峡
□□□□□□□□□□
敵の大艦隊が動けない

ギリシャ艦隊
→ → → 攻撃
```

狭い海峡では大型艦隊が自由に動けず、機動力に優れたギリシャ艦隊が勝利した。

この勝利により、ペルシアはギリシャ征服を断念することになる。

---

## 銀山を利用した海軍建設

アテネ近郊のラウリオン銀山で大量の銀が採掘された。

多くの市民はその利益を分配してほしいと考えていたが、テミストクレスは

> 「今は配るべきではない。軍艦を建造すべきだ。」

と主張した。

この決断によって約200隻の三段櫂船（トリレーム）が建造され、後のサラミスの勝利につながったとされている。

---

## 晩年

戦争の英雄となった後も、アテネでは政争が続いた。

テミストクレスはやがて政敵に敗れ、**陶片追放（とうへんついほう、オストラキスモス）**によってアテネを追放される。

その後、皮肉なことに、かつて戦ったペルシアへ亡命し、ペルシア王のもとで余生を送ったと伝えられている。

---

## 孫子との共通点

[『孫子』](孫子.md)と比べると、テミストクレスの戦い方には共通する考え方が見られる。

| 孫子          | テミストクレス           |
| ----------- | ----------------- |
| 有利な状況を作って戦う | 狭い海峡を戦場に選んだ       |
| 敵の強みを封じる    | ペルシア艦隊の数的優位を無力化した |
| 情報・策略を重視    | 敵をサラミスへ誘い込む策略を用いた |
| 無謀な正面決戦を避ける | 地形を利用して勝利した       |

このように、テミストクレスは「力で押し切る」のではなく、**地形・情報・戦略を活用して勝利した指揮官**として高く評価されている。

そのため、西洋ではしばしば「古代最高の戦略家の一人」として、アレクサンドロス大王やハンニバルなどと並んで語られることがある。
