# クリスティーナ女王

ここでいう**クリスティーナ**は、[スウェーデン帝国](スウェーデン帝国.md)最盛期の女王、**クリスティーナ女王**（1626～1689年）のことだ。
父は「北方の獅子」と呼ばれた名君、**グスタフ2世アドルフ**。スウェーデン史上でも特に異色の君主として知られている。

## 基本情報

| 項目  | 内容               |
| --- | ---------------- |
| 生年  | 1626年            |
| 在位  | 1632～1654年       |
| 王朝  | ヴァーサ朝            |
| 父   | グスタフ2世アドルフ       |
| 後継者 | カール10世グスタフ       |
| 退位後 | カトリックに改宗し、ローマへ移住 |
| 死去  | 1689年、ローマ        |

父王が三十年戦争中の1632年に戦死したため、クリスティーナは**わずか6歳で女王**になった。ただし、幼少期は宰相アクセル・オクセンシェルナを中心とする摂政政府が国政を担い、本人が本格的に統治を始めたのは1644年からだ。

## 学問好きの女王

クリスティーナの最大の特徴は、軍事よりも**学問・哲学・芸術を好んだこと**だ。

幼い頃から王位継承者として教育され、

* ラテン語
* ギリシャ語
* フランス語
* ドイツ語
* 神学
* 哲学
* 歴史
* 政治学

などを学んだ。読書量が多く、ヨーロッパ各地から書物や美術品を集め、ストックホルムを北方の文化都市にしようとした。

フランスの哲学者**ルネ・デカルト**を宮廷に招いたことでも有名だ。デカルトは女王に哲学を教えるためスウェーデンへ赴いたが、厳寒のストックホルムで早朝の講義を行った後に体調を崩し、1650年に亡くなった。

## スウェーデン帝国最盛期の女王

クリスティーナの治世中、スウェーデンは三十年戦争を戦い続けた。

1648年の**ウェストファリア条約**によって、スウェーデンは北ドイツの領土や神聖ローマ帝国内での政治的影響力を獲得した。これによってスウェーデンは名実ともにヨーロッパ列強となった。

つまり、クリスティーナ自身は軍事的な女王ではなかったが、彼女の治世にスウェーデン帝国は国際的な絶頂期へ到達した。

ただし、実際に戦争や外交を主導したのは、オクセンシェルナをはじめとする有力貴族や軍人たちだった。

## 結婚を拒否

クリスティーナは、貴族や議会から何度も結婚を求められた。

女王が結婚して子供を産まなければ、王位継承が不安定になるからだ。しかし本人は結婚を強く拒んだ。

理由については諸説ある。

* 王としての自由を失いたくなかった
* 男性に従属する立場を嫌った
* 学問や政治に集中したかった
* 結婚そのものに関心が薄かった
* カトリックへの改宗を考えていた

従兄のカール・グスタフを後継者に指定し、自分は未婚のまま王位を退く道を選んだ。

## 突然の退位

1654年、クリスティーナは27歳で突然退位した。

後継者には従兄の**カール10世グスタフ**が即位した。

退位の理由は一つではなく、複数の事情が重なっていたと考えられている。

第一に、結婚と出産を要求され続ける王位から離れたかった。第二に、豪華な宮廷生活や文化事業によって財政が悪化していた。第三に、ルター派国家スウェーデンの女王でありながら、密かにカトリックへ傾倒していた。

スウェーデン国王はルター派であることが事実上必須だったため、カトリックへの改宗と王位の維持は両立しなかった。

## カトリックへの改宗

退位後、クリスティーナは男装に近い服装でスウェーデンを離れ、ヨーロッパを南下した。

1654年に秘密裏にカトリックへ改宗し、翌1655年に公表。その後ローマへ入り、教皇アレクサンデル7世に歓迎された。プロテスタントの大国スウェーデンの元女王がカトリックへ改宗したことは、当時のヨーロッパに大きな衝撃を与えた。

これは単なる個人的な宗旨替えではない。

三十年戦争では、父グスタフ2世アドルフがプロテスタント陣営の英雄としてカトリック勢力と戦っていた。その娘がカトリックに改宗したため、象徴的には**父の宗教的遺産を否定したようにも見えた**からだ。

もっとも、クリスティーナ本人は宗派間対立に距離を置き、より普遍的な知性や信仰を求めていたと考えられている。

## ローマでの活動

ローマに移住した後も、静かに隠居したわけではない。

クリスティーナは宮殿に知識人、芸術家、音楽家を集め、文化サロンを形成した。

特に、

* 演劇
* オペラ
* 音楽
* 哲学
* 古典研究
* 美術品収集

を支援した。

女性の舞台出演が制限されることもあったローマで、女性歌手や女優を保護するなど、文化面で大きな役割を果たした。現在では、17世紀ローマの芸術・音楽文化を支えた代表的パトロンの一人と評価されている。

一方で、政治への関心も捨てておらず、ナポリ王位やポーランド王位を狙ったこともあった。しかし、いずれも実現しなかった。

## モナルデスキ処刑事件

クリスティーナの生涯で特に評判を落としたのが、1657年の**モナルデスキ事件**だ。

フランス滞在中、側近ジャン・リナルド・モナルデスキが機密を漏らしたとして、クリスティーナは自ら裁判を行い、宮殿内で処刑させた。

彼女は退位後も自分を主権者と考え、「元女王として家臣を裁く権利がある」と主張した。しかしフランス国内で勝手に処刑を行ったため、強い批判を浴びた。

この事件によって、知的で自由な女王という評価だけでなく、専制的で気まぐれな人物という印象も強くなった。

## 男性的に育てられたのか

クリスティーナはしばしば「男のように育てられた女王」と説明される。

父グスタフ2世アドルフは、娘を単なる王女ではなく、王位継承者として教育するよう命じたとされる。そのため乗馬、狩猟、政治、軍事など、当時は男性君主向けと考えられていた教育も受けた。

本人も豪華な女性用ドレスを好まず、簡素な服装や男性的な振る舞いを見せることがあった。

ただし、現代的な性自認や性的指向をそのまま当てはめるのは難しい。彼女の服装や結婚拒否をめぐってさまざまな解釈があるが、確実に言えるのは、**17世紀の女性君主に期待された役割を強く拒絶した人物**だったということだ。

## 最期

クリスティーナは1689年、ローマで62歳で死去した。

遺体は**サン・ピエトロ大聖堂**に葬られた。女性として、また王族としても非常に例外的な待遇だ。

## 「北方のミネルヴァ」という異名

**「北方のミネルヴァ」**は、クリスティーナの異名だ。

ミネルヴァはローマ神話の**知恵・学問・芸術・戦略**を司る女神で、ギリシャ神話のアテナに相当する。クリスティーナがこの名で呼ばれたのは、北ヨーロッパの君主としては例外的なほど学問と文化を重んじたからだ。

彼女はラテン語や哲学、神学、歴史に通じ、書物・絵画・彫刻を収集し、ヨーロッパ各地の知識人を宮廷へ招いた。特にフランスの哲学者デカルトをストックホルムに招いたことが有名だ。

ただし、この異名は単に「頭のよい女王」というだけではない。

クリスティーナ自身も、知的君主としてのイメージを積極的に演出していた。肖像画や記念メダルでは、ミネルヴァになぞらえられ、軍事国家スウェーデンの支配者でありながら、武力よりも知性や芸術を重んじる君主として表現された。

つまり「北方のミネルヴァ」とは、

> 北ヨーロッパに現れた、知恵と学芸を象徴する女王

という意味だ。

一方で、かなり理想化された呼び名でもある。実際のクリスティーナは文化保護者だった反面、宮廷支出による財政負担や政治的混乱も招いており、完全な名君というわけではなかった。異名は、彼女の実像そのものというより、**彼女が目指した知的君主像**をよく表している。

## どんな人物だったのか

一言で表すなら、

> 帝国を拡大することより、知識と自由を求め、王位そのものを捨てた女王

だ。

父グスタフ2世アドルフが「戦争によってスウェーデン帝国を築いた王」なら、クリスティーナは「帝国の絶頂期に王位を捨て、文化人として生きる道を選んだ女王」だった。

そのため評価は大きく分かれる。

* 教養豊かな文化保護者
* 女性の社会的制約に抗った先駆者
* 国家財政を浪費した君主
* 王としての責任を放棄した人物
* 宗教的・知的自由を求めた異端児

これらがすべて同時に当てはまる、非常に複雑な人物だ。
