# 啓示

**啓示（けいじ）**とは、**これまで知られていなかった真理や知識、意思などが明らかに示されること**を意味する言葉である。

「啓」は「開く・教え導く」、「示」は「示す」という意味があり、「教えを明らかに示すこと」を表している。

## 宗教における啓示

宗教では、啓示とは**神や超越的な存在が人間に教えや真理を伝えること**を指す。

啓示を受けた人は、その内容を人々に伝える役割を担うことが多い。

例えば、

* モーセが十戒を授かったとされる出来事
* ムハンマドが神から『クルアーン』の内容を授かったとされる出来事

などは、代表的な啓示の例である。

キリスト教では、神が歴史やイエス・キリストを通してご自身を人々に示したことも「啓示」と考えられている。

## 一般的な意味

宗教以外では、「隠れていたことが明らかになる」という意味で使われることもある。

例えば、

* 「新しい事実が啓示された」
* 「この本は私にとって人生の啓示だった」

のように、重要な発見や気づきを表す比喩として用いられる。

## 「啓示」と「天啓」の違い

よく似た言葉だが、意味には少し違いがある。

| 啓示               | [天啓](天啓.md)                      |
| ---------------- | ----------------------- |
| 真理や教えが明らかに示されること | 天（神）から授けられる啓示           |
| 示される内容そのものや行為を指す | 神から与えられることを強調する         |
| 宗教以外でも比喩的に使われる   | 「突然のひらめき」の意味でも使われることがある |

つまり、**[天啓](天啓.md)は「天から与えられた啓示」の一種**であり、啓示という言葉のほうがより広い意味を持つ。

## 「啓発」との違い

「啓示」と混同されやすい言葉に「啓発」がある。

* **啓示**：超越的な存在や重要な出来事によって真理が示されること。
* **啓発**：人を教育したり刺激したりして、知識や意識を高めること。

例えば、講演を聞いて知識が深まるのは「啓発」であり、宗教的に神から教えを受けることは「啓示」である。

## 歴史的な意義

啓示という考え方は、多くの宗教の成立と発展において中心的な役割を果たしてきた。

預言者や聖人が受けたとされる啓示は、経典や教義の基礎となり、信仰や倫理、文化に大きな影響を与えてきた。一方で、現代では宗教的な意味だけでなく、「人生を変えるような大きな気づき」を表す比喩としても広く用いられている。
