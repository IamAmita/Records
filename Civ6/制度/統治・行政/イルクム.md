# イルクム

**イルクム（Ilkum）**とは、古代メソポタミア、特にバビロニアで行われていた**国家に対する奉仕義務**の制度を指す。

簡単に言えば、

> **「土地を与えられる代わりに、国のために働いたり軍務に就いたりする義務」**

という仕組みである。

## どのような制度だったのか

古代メソポタミアでは、王が兵士や役人、農民などに土地の使用権を与えることがあった。

その代わりとして、土地を受け取った人には一定の義務が課された。

例えば、

* 軍隊に参加する
* 城壁や運河の建設に従事する
* 灌漑施設の維持管理を行う
* 公共事業に参加する

といった奉仕を行う必要があった。

つまり、土地は自由にもらえるのではなく、「国家への奉仕」と引き換えだった。

## なぜ必要だったのか

メソポタミアでは、

* 大規模な灌漑施設の維持
* 都市の防衛
* 外敵との戦争

が国家の存続に不可欠だった。

そのため、王は税だけでなく、労働力や兵力も安定して確保する必要があった。

イルクム制度は、そのための仕組みだった。

## ハンムラビ法典との関係

[ハンムラビ法典](法典.md)には、イルクムに関する規定が含まれている。

例えば、

* イルクムとして与えられた土地は自由に売却できない。
* 軍務を怠った者への処罰。
* 戦争で捕虜になった兵士の土地や家族の保護。

などが定められており、国家が兵役や公共奉仕を重視していたことが分かる。

## 他の制度との違い

| 制度       | 内容                      |
| -------- | ----------------------- |
| **イルクム** | 土地の給付と引き換えに軍務や労役を行う制度   |
| **[徴兵](../軍事/徴兵.md)**   | 国民を軍隊へ動員する制度            |
| **[賦役](賦役.md)**   | 税や労働を課す制度               |
| **[労働徴発](労働徴発.md)** | 特定の目的のために労働力を強制的に動員すること |

イルクムは単なる兵役制度ではなく、**土地の保有と国家への奉仕が一体となった制度**という点に特徴がある。

## 歴史的な意義

イルクムは、古代国家が

* 軍事力
* 農業生産
* 灌漑事業
* 公共事業

を維持するために生み出した制度の一つだった。

後の時代にも、土地の保有と軍事・行政上の義務を結び付ける制度は各地で見られるようになり、中世ヨーロッパの封建制度などとも共通する考え方がある。ただし、イルクムと封建制度は成立した時代や社会構造が異なるため、同一の制度ではない。
