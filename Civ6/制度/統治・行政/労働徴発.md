# 労働徴発

**労働徴発（ろうどうちょうはつ）**とは、国家や軍、あるいは支配者が、個人の意思にかかわらず強制的に労働へ従事させることをいう。簡単に言えば、**「人を強制的に働かせる制度」**だ。

## どんな目的で行われたのか

歴史上、労働徴発は主に次のような目的で行われた。

* 城壁・城・宮殿の建設
* 道路や橋、運河の建設
* 灌漑施設の整備
* 鉱山での採掘
* 軍事施設の建設
* 戦時中の兵站（補給）業務

特に戦争中には、兵士だけではなく一般市民も塹壕掘りや物資運搬などに動員された。

## 労働徴発と[徴兵](../軍事/徴兵.md)の違い

| 労働徴発       | 徴兵           |
| ---------- | ------------ |
| 強制的に労働させる  | 強制的に軍隊へ入隊させる |
| 土木・建設・運搬など | 戦闘や軍事任務      |
| 一般市民も対象    | 主に兵役対象者      |

どちらも国家による強制動員だが、目的が異なる。

## 世界史での例

### 古代エジプト

[ピラミッド](../../エジプト帝国/歴史/エジプト帝国.md)建設では、奴隷だけでなく、ナイル川の氾濫期で農業ができない農民が国家事業として動員されたと考えられている。

### 古代中国

万里の長城や運河、始皇帝陵などの巨大建築では、多数の農民が労働徴発された。過酷な労働環境から、多くの死者が出たことでも知られる。

### ローマ帝国

道路・水道橋・要塞建設には兵士だけでなく属州住民が労働を課されることもあった。

## 日本での例

日本でも古代から労働徴発に近い制度が存在した。

### 律令国家

農民には税だけでなく、道路や堤防の建設などに従事する**雑徭（ぞうよう）**や**夫役（ぶやく）**と呼ばれる労役が課された。これは税の一種と考えられていた。

### 戦時中

第二次世界大戦では、軍需工場や飛行場建設、防空壕掘削などのために、多くの国民や植民地住民が労働動員された。日本では当時の**徴発令**に基づき、人員だけでなく馬・車両・建物・物資なども軍が徴発できた。

## 労働徴発と奴隷労働の違い

両者は似ているが、必ずしも同じではない。

* **労働徴発**：国家が一定期間、国民などに労働義務を課す制度。
* **奴隷労働**：人間そのものを所有物として扱い、恒常的に労働させる制度。

労働徴発は期間限定であったり、法制度に基づいて行われたりすることが多い一方、奴隷制度では自由そのものが奪われる。

## 現代では

現代の多くの民主国家では、平時の労働徴発は基本的人権や労働の自由に反するため認められていない。

ただし、戦争や大規模災害などの非常事態では、法律に基づいて一定の協力義務や動員が課される場合がある。例えば日本でも、防衛出動時などに物資の収用等を認める制度が自衛隊法に設けられているが、平時に一般国民を強制労働させる制度ではない。
