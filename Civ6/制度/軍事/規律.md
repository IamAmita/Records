# 規律

**規律（きりつ）**とは、**集団や社会の秩序を保つために、守るべきルールや行動の決まり**のことだ。

また、そのルールをきちんと守ることも「規律」と呼ぶ。

## 規律の意味

例えば、学校では

* 授業中は静かにする
* 時間を守る
* 校則を守る

といった決まりがある。

これらは学校生活を円滑にするための**規律**だ。

同じように、会社、軍隊、スポーツチームなどにも、それぞれの規律がある。

## 規律とルールの違い

* **ルール**：個々の決まり。
* **規律**：ルール全体や、それを守って秩序が保たれた状態。

例えば、

* 「9時までに集合する」は一つのルール。
* 「全員が時間を守って行動する」という状態は規律が保たれていると言える。

## 規律が重要な理由

規律があることで、

* 秩序が保たれる
* トラブルが減る
* 公平な行動ができる
* 集団で協力しやすくなる

逆に規律が乱れると、

* 混乱が起きる
* 信頼関係が崩れる
* 組織がうまく機能しなくなる

ことがある。

## 歴史での「規律」

世界史では、特に**軍隊の規律**という形で登場することが多い。

例えば古代ローマ軍は、

* 命令を厳守する
* 隊列を崩さない
* 勝手に戦わない

といった厳しい規律によって、高い戦闘力を維持していた。同様に、[カロライン](../../スウェーデン帝国/歴史/カロライン.md)と呼ばれたスウェーデン軍も、厳しい規律によって精強さを保っていた。

また、[法典](../統治・行政/法典.md)や宗教による規律は、国家や社会を安定させる重要な要素だった。

## 規律と似た言葉

| 言葉     | 意味                       |
| ------ | ------------------------ |
| **規律** | 集団の秩序を保つための決まりや、それを守ること。 |
| **規則** | 個々の具体的な決まり。              |
| **法律** | 国が定める法的拘束力のある規則。         |
| **秩序** | 社会や集団が整った状態。             |

## 例文

* 「軍隊は厳しい**規律**で知られていた。」
* 「学校の**規律**を守る。」
* 「規律ある行動が事故を防ぐ。」

つまり**規律**とは、**集団や社会の秩序を維持するための決まりや、その決まりを守って行動すること**を意味する言葉だ。
